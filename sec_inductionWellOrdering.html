    Section 1.1 Induction and the Well-Ordering Property

    Guiding Questions

    In this section, we’ll seek to answer the following questions:
    🔗
    • What is the Well-Ordering Principle?
      🔗
    • What is mathematical induction, and how can we use it to prove statements about \(\N\text{?}\)
      🔗
    🔗
    In this section we will assume the basic algebraic/arithmetic properties of the integers such as closure under addition, subtraction, and multiplication, most of which we will formalize via axioms in subsequent sections. Axiom 1.1.2 formalizes the familiar notion that nonempty subsets of the positive integers have a smallest element, which will be used repeatedly throughout the text. We then explore a closely related idea, mathematical induction, via an example and exercises.
    🔗

    Subsection 1.1.1 Well-Ordering Principle

    Definition 1.1.1.

    The collection of natural numbers is denoted by \(\N\text{,}\) and is the set
    \begin{equation*} \N = \set{1, 2, 3, \ldots}\text{.} \end{equation*}
    🔗
    By \(\N_0\) we mean the set \(\N\cup \set{0} = \set{0, 1, 2, 3, \ldots}\text{.}\)
    🔗
    🔗
    In some sense, the fundamental properties of \(\N\) are
    • A beginning: there is a smallest natural number
      🔗
    • Succession: there is always a next natural number
      🔗
    🔗

    Aside: Peano Axioms.

    In fact, one can build a model of \(\N\) with set theory and the Peano axioms, which utilize the notion of a successor--the next natural number.
    🔗
    A consequence of the Peano postulates is the well-ordering principle, which we state as an axiom.
    🔗

    Axiom 1.1.2. Well-Ordering Principle.

    Every nonempty subset of \(\N_0\) contains a least (smallest) element under the usual ordering, \(\le\)
     1 
    Our word choice is suggestive. In fact, other orderings do exist, and while the set of nonnegative real numbers \(\mathbb{R}\) does not satisfy the well-ordering principle under the usual ordering \(\le\text{,}\) the Well Ordering Axiom asserts that there exists a well ordering on any set, including \(\mathbb{R}\text{.}\) Accepting this axiom is equivalent to accepting the axiom of choice.
    .
    🔗
    🔗
    The Well-Ordering Principle is useful for producing smallest elements of nonempty subsets defined to have certain properties, as the following example demonstrates.
    🔗

    Exploration 1.1.1.

    In this exploration, we investigate polynomials with real coefficients, as well as their degrees. We will define these terms more formally in Definition 2.2.1, but for now you may use your intuition from previous courses in algebra.
    🔗
    Let \(S\) be the set of all polynomials \(f\) in the variable \(x\) with real coefficients such that \(f(2) = f(-2) = 0\) and \(f(0) = -4\text{.}\)
    🔗
    1. Give an example of an \(f\in S\) and \(g\notin S\text{.}\)
      🔗
    2. Let \(D = \{\deg f\mid f\in S\}\) be the set of possible degrees of polynomials in \(S\text{.}\) Show that \(D\ne \emptyset\) and \(D\subseteq \N_0\text{.}\)
      🔗
    3. Apply the Well-Ordering Principle to argue that \(D\) has a least element. To what does this correspond in \(S\text{?}\)
      🔗
    🔗
    Solution.
    1. Examples vary, but \(f(x) = x^2-4\) and \(g(x) = x+1\) suffice.
      🔗
    2. Since \(f(x) = x^2 - 4\in S\text{,}\) we see that \(2\in D\text{.}\) Furthermore, the zero polynomial is not in \(S\text{,}\) so the degrees of polynomials in \(S\) are all nonnegative integers. Thus, \(D\subseteq \N_0\text{.}\)
      🔗
    3. We see that \(D\) meets the conditions of the well-ordering principle, and so it has a least element. This least element is the degree of the polynomial of least degree satisfying \(f(2) = f(-2) = 0\) and \(f(0) = -4\text{.}\)
      🔗
    🔗
    🔗
    🔗
    We will use this principle throughout the text, next in Theorem 1.2.5.
    🔗

    Definition 1.1.3.

    The set of integers consists of the positive and negative natural numbers, together with zero, and is denoted by \(\Z\text{:}\)
    \begin{equation*} \Z = \set{\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots}\text{.} \end{equation*}
    🔗
    🔗
    🔗

    Subsection 1.1.2 Principle of Mathematical Induction

    Theorem 1.1.4. Principle of Mathematical Induction.

    Let \(P(m)\) be a statement for each \(m\in \N\text{.}\) Suppose that
    1. \(P(1)\) is true.
      🔗
    2. For all \(k\geq 0\text{,}\) if \(P(k)\) is true, then \(P(k+1)\) is true.
      🔗
    Then \(P(m)\) is true for all \(m\in\N_0\text{.}\)
    🔗
    🔗

    Proof.

    Suppose \(P(m)\) is a statement for each \(m\in\N\) satisfying both properties above. Suppose, for a contradiction, that there exists some \(m_0\in\N\) such that \(P(m_0)\) is not true. Let \(S = \{m\in\N\mid P(m)\text{ is not true}\}\text{.}\) Then \(m_0\in S\) by assumption so that \(S\) is a nonempty subset of \(\N\text{.}\) By the Well-Ordering Principle Axiom 1.1.2, \(S\) has a least element.
    🔗
    Let \(k_0\in S\) be the least element. Since \(P(1)\) is true, we know \(k_0> 1\) and so \(k_0-1\in\N\text{.}\) Since \(k_0\) was the least element we know that \(k_0-1\notin S\text{.}\) Thus \(P(k_0-1)\) is true. Since \(P(k_0-1)\) is true, the second hypothesis tells us \(P(k_0)\) is true. This contradicts that \(P(k_0)\) is false. Since we have arrived at a contradiction, we conclude that \(P(m)\) is true for all \(m\in\N\) .
    🔗
    🔗

    Aside: Sample Statements for \(P(m)\).

    Sample statements could include “\(m\) is really interesting” or “\(3m^2 + m + 2\) is even”.
    🔗
    Mathematical induction is like climbing an infinite staircase. The base case tells us that we can take a first step on the staircase. In the inductive hypothesis, we assume we can take all the steps up to a certain height (\(n\)). In the inductive step, we prove that this allows us to take the \((n+1)\)st step.
    🔗
    Thus, if we can take step \(1\text{,}\) we can (by the inductive step) take step \(2\text{.}\) And since we can take step \(2\text{,}\) we can (again by the inductive step) take step \(3 \text{.}\) And so on, forever (or, if the notion of actual infinity makes you uncomfortable, as far as we want to go).
    🔗

    Aside: Induction vs Well-Ordering.

    It can be proved that the principle of mathematical induction is logically equivalent to the well-ordering principle. That is, if we assume either one to be true, we can deduce the other. We will generally take the well-ordering principle on \(\N\) to be an axiom, and treat induction as a theorem.
    🔗
    The following proposition is often referred to as the Gauss Summation Formula. We will employ two proof techniques. The first will be a proof using Theorem 1.1.4 and the second will be a direct proof. In the inductive proof we will introduce some language that is typically used when performing a proof by induction.
    🔗

    Proposition 1.1.5. Gauss Summation Formula.

    For all \(n \ge 1\text{,}\)
    \begin{equation*} 1+ 2 + 3 + \cdots + n = \frac{n(n+1)}{2}\text{.} \end{equation*}
    🔗
    🔗

    Inductive Proof.

    Base case: When \(n = 1\text{,}\) the equation \(1 = \frac{1\cdot (1+1)}{2}\) is true.
    🔗
    Inductive Hypothesis: Assume that there exists a \(n\) such that whenever \(k \le n\text{,}\) the equation
    \begin{equation} 1+ 2 + 3 + \cdots + k = \frac{k(k+1)}{2}\tag{1.1.1} \end{equation}
    is true.
    🔗
    Inductive Step: Our goal is to show that \(P(n+1)\) is true. That is, we wish to establish that
    \begin{equation} 1+ 2 + 3 + \cdots + n + (n+1) = \frac{(n+1)((n+1)+1)}{2}\text{.}\tag{1.1.2} \end{equation}
    🔗
    We begin on the left-hand side of (1.1.1), where we may apply the inductive hypothesis to see that
    \begin{equation} 1+ 2 + 3 + \cdots + n + (n+1) = \left[\frac{n(n+1)}{2}\right] + (n+1)\text{.}\tag{1.1.3} \end{equation}
    Through the use of straightforward algebra, the right-hand side becomes
    \begin{equation} \frac{n(n+1)}{2} + \frac{2(n+1)}{2} = \frac{n(n+1) + 2(n+1)}{2} = \frac{(n+1)(n+2)}{2}\tag{1.1.4} \end{equation}
    .
    🔗
    Putting (1.1.3) and (1.1.4) together, we obtain
    \begin{equation*} 1+ 2 + 3 + \cdots + n + (n+1) = \frac{(n+1)((n+1)+1)}{2}\text{,} \end{equation*}
    which is exactly the goal we stated in (1.1.2).
    🔗
    🔗

    Direct Proof.

    Let \(n\in\N\) and consider the summation
    \begin{equation*} S = 1+2+3+\cdots +(n-2)+ (n-1) + n \end{equation*}
    and the same summation written in reverse:
    \begin{equation*} S = n+(n-1)+(n-2) + \cdots + 3 + 2 + 1. \end{equation*}
    Add these together termise:
    \begin{equation*} 2S = (n+1) + (n+1) + (n+1)+\cdots + (n+1)+(n+1) + (n+1). \end{equation*}
    There are \(n\) total terms so we arrive at \(2S = n(n+1)\) and so \(S = \frac{n(n+1)}{2}\text{.}\) Thus
    \begin{equation*} 1+2+3+\cdots + n = \frac{n(n+1)}{2}. \end{equation*}
    🔗
    🔗
    We choose to give two proofs of the Gauss Summation Formula to illustrate that induction is not always the easiest way to prove statements about natural numbers. However, it is a crucial tool in algebra and it’s important to learn how to use our tools.
    🔗
    With the goal of understanding how to use our tools, we continue with another example of induction. This example shows how to manipulate inequalities within proofs.
    🔗

    Proposition 1.1.6.

    For all \(k\ge 1\text{,}\) \(3^k > k\text{.}\)
    🔗
    🔗

    Proof.

    Let’s start by labelling \(P(m)\text{.}\) By looking at the statement, \(P(m)\) should be:
    \begin{equation*} P(m)\colon 3^m>m. \end{equation*}
    🔗
    Now that \(P(m)\) is clearly labelled, we will prove the base case which is \(P(1)\text{.}\) With our description, we have \(P(1)\) is the statement \(3^1>1\) which is certainly true. This completes the proof of the base case.
    🔗
    For the inductive step, we suppose \(P(k)\) is true and prove that \(P(k+1)\) is also true. Given our description of the statements \(P(m)\text{,}\) assuming \(P(k)\) is true is assuming
    \begin{equation*} 3^k>k. \end{equation*}
    We now need to use this to show that \(P(k+1)\) is true. That is, we need to show \(3^{k+1}>k+1\text{.}\) Before we perform computations let’s also notice that \(k\ge 1\text{.}\) Now we compute:
    \begin{align*} 3^{k+1} \amp = 3^k\cdot 3^1\\ \amp > k\cdot 3\\ \amp = k + 2k \\ \amp > k + 1 \end{align*}
    where the last inequality follows from \(k\ge 1\) implies \(2k\ge 2>1\text{.}\) We conclude \(3^{k+1}>k+1\) as desired.
    🔗
    This completes the proof by induction and we conclude \(3^m>m\) for all \(m\in \N\) .
    🔗
    🔗

    Theorem 1.1.7. Sum of Cubes Formula.

    Prove that the sum of the first \(n\) cubes is \(\dfrac{n^2(n+1)^2}{4}\text{.}\) That is,
    \begin{equation*} 1^3 + 2^3 + 3^3 + \cdots + n^3 = \dfrac{n^2(n+1)^2}{4}. \end{equation*}
    🔗
    🔗

    Inductive Proof.

    We wish to prove that
    \begin{equation*} 1^3 + 2^3 + 3^3 + \cdots + n^3 = \frac{n^2(n+1)^2}{4}. \end{equation*}
    🔗
    Base Case: When \(n=1\text{,}\) the theorem clearly holds.
    🔗
    Inductive Hypothesis: Assume for all \(k\le n\) that we have
    \begin{equation} 1^3 + 2^3 + 3^3 + \cdots + k^3 = \frac{k^2(k+1)^2}{4}.\tag{1.1.5} \end{equation}
    🔗
    Inductive Step: We wish to prove that
    \begin{equation*} 1^3 + 2^3 + 3^3 + \cdots + n^3 +(n+1)^3 = \frac{(n+1)^2((n+1)+1)^2}{4} = \frac{(n+1)^2(n+2)^2}{4}. \end{equation*}
    We apply (1.1.5) to obtain
    \begin{align*} 1^3 + 2^3 + 3^3 + \cdots + n^3 +(n+1)^3 \amp = \frac{n^2(n+1)^2}{4} + (n+1)^3 \\ \amp = \frac{n^2(n+1)^2}{4} + \frac{4(n+1)^3}{4}\\ \amp = \frac{n^2(n+1)^2 + 4(n+1)^3}{4}\\ \amp = \frac{(n+1)^2 (n^2+4(n+1))}{4}\\ \amp = \frac{(n+1)^2 (n^2+4n+4)}{4}\\ \amp = \frac{(n+1)^2 (n+2)^2}{4}. \end{align*}
    The theorem is proved.
    🔗
    🔗

    Theorem 1.1.8. (Bernoulli’s Inequality).

    Given a real number \(b > -1\text{,}\) \((1+b)^n \ge 1 + bn\) for all \(n\in \N_0\) .
    🔗
    🔗

    Proof.

    When \(n=0\) the inequality becomes \(1 \ge 1\text{,}\) which is clearly true.
    🔗
    Assume \(n > 0\text{.}\) Then \((1+b)^{n+1} = (1+b)(1+b)^{n+1} \ge (1+b)(1+b)^n = 1 +bn + b+b^2 n = 1 + b(n+1) + b^2 n \ge 1+b(n+1)\text{.}\)
    🔗
    🔗
    🔗
