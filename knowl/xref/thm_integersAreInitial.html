Theorem 3.1.11.

For any ring \(R\text{,}\) there is a unique homomorphism \(\varphi\colon \Z\to R\text{.}\)

Proof.

Let \(R\) be any ring. We need to show two things: there exists a homomorphism from \(\Z\) and there is only one such homomorphism.
For constructing the homomorphism, notice that if it existed it must preserve the unity. So we begin defining \(\varphi\colon \Z\to R\) by \(\varphi(1) = 1_R\text{.}\) We then proceed to inductively define \(\varphi\) for each \(n\in\) by \(\varphi(n+1) = \varphi(n)+\varphi(1)\text{.}\)
