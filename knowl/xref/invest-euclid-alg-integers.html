Investigation 1.2.6. (Euclidean Algorithm).

Let \(a,b\in \N\text{.}\) Use Theorem 1.2.5 and Investigation 1.2.5 to determine an algorithm for computing \(\gcd(a,b)\text{.}\) How could your method be modified to compute \(\gcd(a,b)\) for \(a,b\in\Z\text{?}\)
Solution.
Let \(a,b\in \N\text{.}\) Write \(a = bq_1+r_1\text{.}\) By Theorem 1.2.5, \(\gcd(a,b) = \gcd(b,r_1)\text{.}\)
Then, write \(b = r_1 q_2 + r_2\) using the Division algorithm. Observe \(\gcd(b,r_1) = \gcd(r_1,r_2)\text{.}\) Continue until there is a \(k\) for which \(r_k = 0\text{,}\) so \(\gcd(r_k,r_{k-1}) = r_{k-1}\text{.}\) Then \(r_{k-1}= \gcd(r_k, r_{k-1}) = \gcd(r_{k-1},r_{k-2}) = \cdots = \gcd(a,b)\text{.}\)
