Proof.

Pick \(r\in R\) and \(x\in\ker\varphi\) so that \(\varphi(x)=0\text{.}\) Then
\begin{equation*} \varphi(rx) = \varphi(r)\varphi(x) = \varphi(r)\cdot 0 = 0. \end{equation*}
Thus \(rx\in \ker\varphi\text{.}\) The proof of \(xr\in\ker\varphi\) is similar and is left as an activity.

\begin{equation*} \varphi(x+y) = \varphi(x)+\varphi(y) = 0 + 0 = 0. \end{equation*}
Thus \(x+y\in\ker\varphi\text{.}\)
