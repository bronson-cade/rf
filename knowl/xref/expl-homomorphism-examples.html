Exploration 3.1.1.

Determine whether the following functions are homomorphisms, monomorphisms, epimorphisms, isomorphisms, automorphisms, or none of these. Note that \(R\) denotes an arbitrary ring and \(F\) a field.
  1. \(\varphi\colon R\to R\) defined by \(\varphi(x)=x\)
  2. \(\psi\colon R\to R\) defined by \(\psi(x)=-x\)
  3. \(\alpha\colon \Z\to \Z\) defined by \(\alpha(x)=5x\)
  4. \(\mathrm{Fr}\colon \mathbb{F}_2[x]\to \mathbb{F}_2[x]\) defined by \(\mathrm{Fr}(p) = p^2\)
  5. \(\iota\colon \mathbb{C}\to \mathbb{C}\) defined by \(\iota(a+bi)=a-bi\text{,}\) where \(a,b\in \R, i^2 = -1\)
  6. \(\beta\colon\Z\to \Z/5\) defined by \(\beta(x) = \overline{x}\)
  7. \(\epsilon_r\colon F[x] \to F\) defined by \(\epsilon_r(p(x)) = p(r)\text{,}\) where if \(p(x) = a_0 + a_1 x + \cdots + a_n x^n\text{,}\) \(p(r)\) is the expression obtained by “plugging \(r\) into \(p\)”: \(p(r) = a_0 + a_1 r + \cdots + a_n r^n\) (this is known as the \(r\)-evaluation map)
  8. \(\xi\colon \Z/5 \to \Z/10\) defined by \(\xi(\overline{x}) = \overline{5x}\)
Solution.
  1. \(\varphi\colon R\to R\) defined by \(\varphi(x)=x\) is called the identity mapping. We in particular have: \(\varphi(1_R) = 1_R\text{,}\) \(\varphi(x+y) = x+y = \varphi(x)+\varphi(y)\text{,}\) and \(\varphi(xy)=xy=\varphi(x)\varphi(y).\) The identity mapping satisfies the criteria for being a ring homomorphism. It is an automorphism.
  2. \(\psi\colon R\to R\) defined by \(\psi(x)=-x\) is the negation mapping. Although it is true that
    \begin{equation*} \psi(x+y) = -(x+y) = -x+(-y)=\psi(x)+\psi(y) \end{equation*}
    it fails the other two properties of a ring homorphism in general (sometimes not though!). Namely, for a general ring \(1_R\neq -1_R\) so that \(\psi(1_R) = -1_R\neq 1_R\) and also \(\psi(xy) = -xy\) but \(\psi(x)\psi(y) = (-x)(-y) = xy\text{.}\) In general, \(xy\neq -xy\) so \(\psi\) is not compatible with multiplication or unity. However, sometimes \(-a = a\) is true for every element of the ring, particularly Boolean rings. Examples include \(\Z/2, \Z/2[x]\text{.}\)
  3. \(\alpha\colon \Z\to \Z\) defined by \(\alpha(x)=5x\text{,}\) or the multiplication by \(5\) map. The map \(\alpha\) does not preserve the unity nor does it preserve multiplication. Check! It is not a ring homomorphism.
  4. \(Fr\colon \Z_2[x]\to \Z_2[x]\) defined by \(Fr(p) = p^2\) is sometimes called the Frobenius mapping. It is in fact a ring homomorphism in this case. For the unity and multiplicative compatibility we have \(Fr(1) = 1^2 = 1\) and \(Fr(pq) = (pq)^2 = p^2q^2=Fr(p)Fr(q)\text{.}\) For compatibility with addition we remind the reader that \(2\equiv_20\text{:}\)
    \begin{align*} Fr(p+q) \amp= (p+q)^2\\ \amp= p^2+2pq+q^2 \\ \amp= p^2+q^2 \\ \amp= Fr(p)+Fr(q). \end{align*}
    It is not an automorphism; however, it is a homomorphism. Homomorphisms from a ring to itself are called endomorphisms.
