Theorem 4.2.10.

Let \(R\) be a PID and let \(p\in R\text{.}\) Then \(p\) is prime if and only if \(p\) is irreducible.

Proof.

Assume that \(p\) is prime. Suppose that \(p=ab\) for some \(a,b \in R\text{.}\) Then \(p\) divides \(ab\) which implies that \(p\) divides \(a\) or \(p\) divides \(b\text{.}\) WLOG, assume that \(p\) divides \(a\text{.}\) Then there exists \(c\in R\) such that \(a=pc\) which implies that \(p=pcb\text{.}\) Therefore \(cb=1\) and \(b\) is a unit which implies that \(p\) is irreducible.

Assume that \(p\) is irreducible. Suppose that \(p\) divides \(ab\) for some \(a,b \in R\) and that \(p\) does not divide \(a\text{.}\) Then \(\ideal{a,p}=R\) and there exists \(x,y\in R\) such that \(1=ax+py\text{.}\) Multiplying boths sides by \(b\) yiels \(b=abx+pby=p(cx+by)\) which implies that \(p\) divides \(b\) and therefore \(p\) is prime.
