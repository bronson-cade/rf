Theorem 3.4.2.

The ring \(\Z\) is a principal ideal domain.

Proof.

The zero ideal is principally generated by \(0\) so we let \(I\subseteq\Z\) be a non-zero ideal and show it is principally generated.
ðŸ”—
Consider \(S=I\cap \mathbb{N}\text{.}\) Since \(I\neq \{0\}\text{,}\) there is \(0\neq a\in I\text{.}\) If \(a\ge 0\text{,}\) then \(a\in S\text{.}\) Otherwise \(-1\cdot a = -a\ge 0\) so that \(-a\in S\) (ideals are closed under multiplication). Thus \(S\) is a non-empty subset of \(\mathbb{N}\) and so it has a least element \(d\text{.}\) We claim \(I = (d)\text{.}\)
ðŸ”—
Take any other element \(b\in I\text{,}\) which we assume is positive. Then the division algorithm tells us there exists \(q,r\in\N\) with \(0\leq r \lt d\) such that \(b = dq+r\text{.}\) But this would tell us \(r = b-dq\in I\) and \(r \lt d\text{.}\) Since \(d\) is the least positive element of \(I\text{,}\) this can only happen if \(r=0\) and so \(b=dq\) and \(b\in (d)\text{.}\) Therefore \(I=(d)\text{.}\)
ðŸ”—
We conclude that \(\Z\) is a PID.
