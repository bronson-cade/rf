Example 3.3.20. Algebraic Definition of Complex Numbers.

At this point, we’ve seen two different incarnations of the complex numbers: Definition 2.1.3 and Exercise 2.1.7.3 We will show yet another way of constructing them.
Let \(\mathrm{ev}_i\colon\R[x]\to\C\) be the evaluation at \(i\) homomorphism. Let’s also note that \(\mathrm{ev}_i(a+bx) = a+bi\text{.}\) Thus \(\mathrm{ev}_i\) is an epimorphism.
Since \(\mathrm{ev}_i(x^2+1) = i^2+1 = -1+1 = 0\) we know that \(x^2+1\in\ker\mathrm{ev}_i\text{.}\) Suppose that \(p(x)\in\ker\mathrm{ev}_i\text{.}\) Using the Polynomial Division Algorithm, we can divide \(p(x)\) by \(x^2+1\) to get
\begin{equation*} p(x) = q(x)(x^2+1) + r(x) \end{equation*}
where \(\deg(r(x)) \lt 2\text{.}\) Since \(p(x)\) is in the kernel we have:
\begin{align*} 0 \amp= p(i) \\ \amp=q(i)(i^2+1)+r(i) \\ \amp=r(i) \end{align*}
and so \(r(i)=0\text{.}\) Since \(r(x)\) has degree at most one, we can write \(r(x) = a+bx\text{.}\) But then
\begin{equation*} 0=r(i) = a+bi \end{equation*}
implies \(a=0,b=0\text{.}\) Thus \(r(x) = 0\) and \(p(x) = q(x)(x^2+1)\text{.}\) In particular, \(p(x)\in (x^2+1)\text{.}\) Finally, we can conclude that \(\ker\mathrm{ev}_i = (x^2+1)\) and by the Noether Isomorphism Theorem
\begin{equation*} \R[x]/(x^2+1)\cong \C. \end{equation*}
This example shows we could redefine the complex numbers as \(\R[x]/(x^2+1)\) and this is a purely algebraic definition of \(\C\) (up to using \(\R\)).
