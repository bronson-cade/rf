Theorem 1.2.3.

Let \(a,b,c\in\Z\text{.}\) If \(a\mid b\text{,}\) then \(a\mid bc\text{.}\)

Proof.

Our hypotheses means there is an integer \(k\) for which \(b = ak\text{.}\) Then \(bc = (ak)c = a(kc)\text{.}\) We observe that \(k' = kc\) is an integer, so \(bc = ak'\text{,}\) and therefore \(a\mid bc\text{.}\)
