Theorem 3.1.12.

For any ring \(R\text{,}\) there is one and only one homomorphism \(\varphi\colon R\to \{0\}\text{.}\)

Proof.

Suppose \(R\) is a ring. There is only one set-theoretic function \(\varphi\colon R\to\{0\}\text{:}\) \(\varphi(r) = 0\) for all \(r\in R\text{.}\) We must show this function is a homomorphism. Recall that in the zero ring, we have \(0+0=0\text{,}\) \(0\cdot 0 = 0\text{,}\) and both the multiplicative and additive identities are \(0\text{.}\)
All of the checks we would need to do involve asking if two elements in the codomain are equal. But there is only one element in the codomain, so the checks go through without any work. The details are left as an activity.
