Example 2.1.14. Adjoining \(\sqrt{-5}\) to the Rationals.

Define the set of rationals adjoin \(\sqrt{-5}\) by:
\begin{equation*} \Q(\sqrt{-5}) = \{a+b\sqrt{-5}\mid a,b\in\mathbb{Q}\}. \end{equation*}
This is a subset of \(\mathbb{C}\text{.}\) We claim that \(\Q(\sqrt{-5})\) is a subfield of \(\mathbb{C}\)
ðŸ”—
We apply the Subfield Test. In particular, \(\mathbb{Q}(\sqrt{-5})\subset \C\) and the latter is a field. We first show that \(1+0i\in\Q(\sqrt{-5})\) but this is just the element \(1+0\sqrt{d}\in\Q(\sqrt{-5})\text{.}\)
ðŸ”—
We now show closure under subtraction. Take \(a+b\sqrt{-5},c+d\sqrt{-5}\in\Q\text{,}\) then
\begin{equation*} (a+b\sqrt{-5}) - (c+d\sqrt{-5}) = (a-c) + (b-d)\sqrt{-5}. \end{equation*}
Since \((a-c),(b-d)\in \Q\text{,}\) we conclude that \(\Q(\sqrt{-5})\) is closed under subtraction.
ðŸ”—
We now show closure under division. Suppose \(a+b\sqrt{-5},c+d\sqrt{-5}\in\Q(\sqrt{-5})\text{.}\) Writing \(c+d\sqrt{-5} = c+d\sqrt{5}i\text{,}\) we know that
\begin{equation*} (c+d\sqrt{5}i)^{-1} = \frac{c}{c^2+5d^2} + \frac{-d\sqrt{5}}{c^2+5d^2}i \end{equation*}
which we can recombine the \(i\) and \(\sqrt{5}\text{:}\)
\begin{equation*} (c+d\sqrt{5}i)^{-1} = \frac{c}{c^2+5d^2} + \frac{-d}{c^2+5d^2}\sqrt{-5}. \end{equation*}
Note that \(c^2+5d^2=0\) would say \((c/d)^2 = -5\) which is absurd so the inverse is well-defined.
ðŸ”—
Now we divide:
\begin{align*} \amp(a+b\sqrt{-5})\cdot (c+d\sqrt{-5})^{-1} \\ =\amp(a+b\sqrt{-5})\cdot \left(\frac{c}{c^2+5d^2} + \frac{-d}{c^2+5d^2}\sqrt{-5} \right) \\ =\amp \left(\frac{ac}{c^2+5d^2} + \frac{5bd}{c^2+5d^2}\right) + \left(\frac{-ad}{c^2+5d^2}+ \frac{bc}{c^2+5d^2}\right)\sqrt{-5}\\ =\amp \frac{ac+5bd}{c^2+5d^2} + \frac{-ad+bc}{c^2+5d^2}\sqrt{-5} \end{align*}
and since \(\Q\) is a field it is closed under arithmetic. In particular \(\frac{ac+5bd}{c^2+5d^2}, \frac{-ad+bc}{c^2+5d^2}\in \Q\text{.}\) Thus \(\Q(\sqrt{-5})\) is closed under division.
ðŸ”—
We conclude, by the Subfield Test, that \(\Q(\sqrt{-5})\) is a subfield of \(\C\text{.}\)
