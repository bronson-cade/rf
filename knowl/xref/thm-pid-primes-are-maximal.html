Theorem 3.4.18.

In a principal ideal domain, every prime ideal is maximal.

Proof.

Let \(R\) be a PID and \(\mathfrak{p}=(p)\) a prime ideal. By previous work, \(p\) is prime. Suppose that \(\mathfrak{p} \subseteq \mathfrak{m}\text{.}\) Thus, \(p\in (m)\text{,}\) so \(m|p\text{.}\) That is, \(p = mk\text{.}\) Since \(p\) is prime and \(R\) is a domain, it is irreducible. Thus, either \(m\) or \(k\) is a unit. If \(m\) is a unit, then \((m) = R\text{.}\) If \(k\) is a unit, then \(m = k^{-1} p\text{,}\) and thus \(m\in (p)\text{,}\) which means that \((m) = (p)\text{.}\)

Thus, \((p)\) is maximal.
