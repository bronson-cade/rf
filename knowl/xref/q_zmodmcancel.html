Investigation 1.4.8.

Let \(\overline{a},\overline{b},\overline{c}\in \Z/m\) with \(\overline{c}\ne\overline{0}\) and \(m > 1\text{.}\) If \(\overline{a}\cdot \overline{c} = \overline{b}\cdot \overline{c}\text{,}\) is it true that \(\overline{a} = \overline{b}\text{?}\) If so, prove it. If not, find an example of when the statement fails to hold.
Answer.
It is not true. For example, in \(\Z/{12}\text{,}\) \(\overline{6}\cdot \overline{3} = \overline{6}\cdot\overline{1}\text{,}\) but \(\overline{3}\ne \overline{1}\text{.}\)
