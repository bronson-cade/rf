Theorem 3.2.12.

Given a commutative ring with identity \(R\) and a nonempty subset \(S\) of \(R\text{:}\)
  1. The set \(I = \setof{r_1 s_1 + r_2 s_2 + \cdots + r_n s_n}{r_i\in R, \ s_j \in S,\ n\ge 1}\) is an ideal of \(R\text{;}\)
  2. \(S\subseteq I\text{;}\) and
  3. \(I = \ideal{S}\text{.}\)

Proof.

It is clear that \(0\in I\text{,}\) and that if \(r\in R\) and \(x\in I\text{,}\) \(rx\in I\text{.}\) Moreover, the sum of two \(R\)-linear combinations of elements of \(S\) is yet another \(R\)-linear combination of elements of \(S\text{.}\) Thus, \(I\) is an ideal. Further, if \(s\in S\text{,}\) \(1\cdot s \in I\text{,}\) so \(S\subseteq I\text{.}\) Therefore, \(\ideal{S}\subseteq I\text{.}\)
Now assume that \(x\in I\text{.}\) Then \(x\) has the form \(x = r_1 s_1 + \cdots + r_n s_n\text{.}\) Each \(s_i \in S\text{,}\) so each \(r_i s_i \in J\) if \(J\) is any ideal containing \(S\text{.}\) In particular, \(r_i s_i \in \ideal{S}\text{,}\) and thus the sum \(x = \sum r_i s_i \in \ideal{S}\text{.}\)
