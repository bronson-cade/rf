Definition 3.1.1.

Let \(R\) and \(S\) be rings. A function \(\varphi : R\to S\) is called a ring homomorphism if it preserves addition, multiplication, and sends the identity of \(R\) to the identity of \(S\text{.}\) That is, for all \(x,y\in R\text{:}\)
  • \(\varphi(x+y) = \varphi(x) + \varphi(y)\text{,}\)

If \(\varphi\) is injective (one-to-one), we say that \(\varphi\) is a monomorphism. If \(\varphi\) is surjective (onto), we say that \(\varphi\) is an epimorphism. If \(\varphi\) is a bijection (both injective and surjective), we say that \(\varphi\) is an isomorphism and write \(R\cong S\text{.}\) If \(\varphi \colon R\to R\) is an isomorphism, we say \(\varphi\) is an automorphism of \(R\text{.}\)
