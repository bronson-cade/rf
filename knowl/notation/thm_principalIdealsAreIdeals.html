Theorem 3.2.9.

Let \(R\) be commutative with identity and let \(a\in R\text{.}\) The set
\begin{equation*} (a) = \{ra\mid r\in R\} \end{equation*}
is an ideal (called the principal ideal generated by \(a\)).

Proof.

Observe that \(0 = 0\cdot a \in (a)\text{.}\)
Moreover, if \(r_1 a, r_2 a\in (a)\text{,}\) \(r_1 a - r_2 a = (r_1 - r_2)a \in (a)\text{.}\) Finally, if \(x\in R\) and \(r a\in (a)\text{,}\) \(x (ra) = (xr) a \in (a)\text{.}\)
Thus, \((a)\) is an ideal.
