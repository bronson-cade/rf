Definition 3.3.6.

Given an ideal \(I\subseteq R\) in a commutative ring \(R\text{.}\) A coset of \(I\) is a set of the form
\begin{equation*} a+I = \{a+i\mid i\in I\}. \end{equation*}
The collection of all cosets is denoted \(R/I\) or \(\bar{R}\) if \(I\) is understood:
\begin{equation*} R/I = \{a+I\mid a\in R\}. \end{equation*}
