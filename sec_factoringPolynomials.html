    Section 4.1 Factoring Polynomials

    Guiding Questions

    In this section, we‚Äôll seek to answer the questions:
    üîó
    • What is an irreducible polynomial? Are there any tools we can use to determine if a given polynomial is irreducible?
      üîó
    üîó
    One of the most beautiful consequences of an abstract study of algebra is the fact that both \(\Z\) and \(F[x]\) are Euclidean domains. While they are not ‚Äúthe same‚Äù, we can expect them to share many of the same properties. In this section, our first goal will be to extend familiar properties from \(\Z\) to \(F[x]\text{.}\) We will also see that particular features of a polynomial (e.g., its degree, or the existence of roots) allows for additional criteria for its irreducibility to be decided.
    üîó
    Since both \(\Z\) and \(F[x]\) have a division algorithm, it is reasonable to expect that, similar to the integers, we can also investigate the greatest common divisor of polynomials. In fact, our method for finding the greatest common divisor of two integers extends nicely to polynomials.
    üîó

    Subsection 4.1.1 Polynomials As Functions

    One of the most useful things we can do with polynomials is evaluate them by ‚Äúplugging in‚Äù elements from our coefficient set (or some superset that contains it) and performing the resulting arithmetic in an appropriate ring. We can make this completely rigorous using the language of functions: given a commutative ring \(R\) and all polynomials \(p(x) = a_0 + a_1 x + \cdots + a_n x^n \in R[x]\text{,}\) we define the function \(p_f : R\to R\) by \(p_f(r) = a_0 + a_1 r + \cdots + a_n r^n \text{.}\) However, we will not belabor this point; instead, we will generally write \(p(r)\) in place of \(p_f(r)\) and appeal to our common notions of evaluating polynomials.
    üîó
    Given a polynomial \(p(x)\in R[x]\text{,}\) we have frequently been interested in finding all \(r\in R\) for which \(p(r) = 0\text{.}\)
    üîó
    We next make the somewhat pedantic ‚Äî yet valuable ‚Äî distinction between the polynomial \(p(x)\in R[x]\) and the function \(p_f : R\to R\) defined by the polynomial.
    üîó

    Definition 4.1.1.

    Let \(R\) be a commutative ring. For any \(p(x)\in R[x]\text{,}\) the corresponding polynomial function \(p_f\colon R \to R\) is defined, for all \(r\in R\) by \(p_f(r) = p(r)\text{.}\) We usually denote \(p_f(r)\) by \(p(r)\text{,}\) and say that \(p(r)\) is the polynomial \(p\) evaluated at \(r\text{.}\)
    üîó
    üîó
    The following example illustrates that there is not, in general, a one-to-one correspondence between polynomials and polynomial functions. This means we must be very careful in identifying polynomials with a polynomial function.
    üîó

    Example 4.1.2. Polynomials vs Polynomial Functions.

    Consider the polynomial ring \(\mathbb{F}_2[x]\) over \(\mathbb{F}_2\text{.}\) We have, in particular, the polynomials \(x^n\in\mathbb{F}_2[x]\) for each \(n\in\mathbb{N}\text{.}\) So the number of polynomials over \(\mathbb{F}_2\) is (countably) infinite.
    üîó
    By contrast, the number of functions \(\mathbb{F}_2\to\mathbb{F}_2\) is far less! There are four functions: the two constant functions, the identity, and the transposition. In particular, the assignment \(p(x)\in\mathbb{F}_2[x]\) to \(p_f\colon \mathbb{F}_2\to\mathbb{F}_2\) is not injective. However, it is surjective (exercise).
    üîó
    By contrast, if we consider polynomials over \(\R\text{.}\) The assignment \(p(x)\in\R[x]\) to a polynomial function \(p_f\colon \R\to \R\) is injective but not surjective. There are more functions than polynomial functions. There are even more smooth functions than polynomial functions.
    üîó
    üîó

    Definition 4.1.3.

    Let \(R\) be commutative with identity and suppose \(p(x) \in R[x]\text{.}\) We say \(r\in R\) is a zero or root of \(p(x)\) if \(p(r) = 0\text{.}\)
    üîó
    üîó
    When considering polynomials with integer coefficients, any rational roots are particularly well-behaved.
    üîó

    Theorem 4.1.4. Roots and Linear Factors.

    Let \(R\) be a commutative ring. For any polynomial \(p(x)\in R[x]\) and \(a\in R\text{,}\) there exists a unique polynomial \(q(x)\in R[x]\) with
    \begin{equation*} p(x) = (x-a)q(x)+p(a). \end{equation*}
    In particular, \(a\) is a root of \(p(x)\) if and only if \((x-a)\) is a factor of \(p(x)\text{.}\)
    üîó
    üîó

    Proof.

    By the division algorithm, there exists unique \(q(x),r(x)\in R[x]\) with
    \begin{equation*} p(x) = (x-a)q(x) + r(x) \end{equation*}
    where \(\deg(r)\lt\deg(x-a)=1\text{.}\) Thus \(r(x) = r\) is constant. Since evaluation is a homomorphism we can evaluate at \(a\text{:}\)
    \begin{equation*} p(a) = (a-a)q(a)+r = r. \end{equation*}
    This completes the first part.
    üîó
    If \(a\) is a zero of \(p(x)\text{,}\) then \(r=0\text{.}\) If \((x-a)\) is a factor of \(p(x)\text{,}\) then \(p(x)=(x-a)q(x)\) and so
    \begin{equation*} p(a) =(a-a)q(a) = 0. \end{equation*}
    So \(a\) is a root.
    üîó
    üîó

    To simplify matters, we will focus henceforth on polynomials with coefficients in a field. The following theorem is a result that you learned in high school algebra (and have likely used countless times since then), but as with the other familiar topics we have explored so far, it is necessary to formalize prior to continuing.
    üîó

    Theorem 4.1.6. Factor Theorem.

    Let \(F\) be a field, and \(p(x)\in F[x]\text{.}\) Then \(\alpha\in F\) is a root of \(p(x)\) if and only if \(x-\alpha\) divides \(p(x)\text{.}\)
    üîó
    üîó

    Proof.

    If \(x-\alpha\) divides \(p(x)\text{,}\) then \(p(x) = q(x) (x-\alpha)\) and \(p(\alpha) = q(\alpha) (\alpha - \alpha) = 0\text{,}\) so \(\alpha\) is a root.
    üîó
    Otherwise, use the division algorithm to divide \(p(x)\) by \(x-\alpha\text{.}\) Then \(p(x) = q(x)(x-\alpha) + r\text{,}\) where \(\deg r \lt \deg (x-\alpha) = 1\text{.}\) Thus, \(r\) is a nonzero constant. If \(\alpha\) is a root of \(p\text{,}\) then \(0 = p(\alpha) = q(\alpha) (\alpha - \alpha) + r = 0 + r = r\text{,}\) so \(r=0\) and \(x-\alpha | p(x)\text{.}\)
    üîó
    üîó
    Note that while \(F[x]\) is a ring, and we already have a definition of an irreducible element of a ring, we will find it useful to have a ready definition of irreducible in the context of polynomials with coefficients in a field. It is to that task that we now turn.
    üîó

    üîó
    üîó

    Definition 4.1.7.

    A polynomial \(f(x)\in F[x]\) is reducible if it is not irreducible.
    üîó
    üîó

    Exploration 4.1.4.

    State a positive definition for a reducible polynomial with coefficients in a field \(F\text{.}\) That is, state a definition which does not refer to the notion of irreducibility.
    üîó
    Solution.
    A polynomial \(r(x)\) is reducible if it can be written as \(r(x) = s(x) t(x)\text{,}\) where \(1 \le \deg s(x) \lt \deg r(x)\) and \(1 \le \deg t(x) \lt \deg r(x)\text{.}\)
    üîó
    üîó
    üîó

    Theorem 4.1.8.

    Every polynomial of degree 1 in \(F[x]\) is irreducible.
    üîó
    üîó

    Proof.

    Let \(f(x)\) be degree 1 and write \(f(x) = s(x) t(x)\text{.}\) Then \(\deg f = 1 = \deg s + \deg t\text{.}\) Since \(\deg s, \deg t \ge 0\text{,}\) one of \(\deg s\) or \(\deg t\) is 0, hence \(s\) or \(t\) is constant.
    üîó
    üîó

    Theorem 4.1.9.

    A nonconstant polynomial \(f(x)\in F[x]\) of degree 2 or 3 is irreducible over \(F\) if and only if it has no zeros in \(F\text{.}\)
    üîó
    üîó

    Proof.

    We prove the double contrapositive: \(f(x)\in F[x]\) of degree 2 or 3 is reducible if and only if it has a zero in \(F\text{.}\)
    üîó
    If \(f(x)\) is reducible there exist nonconstant \(s(x),t(x)\in F[x]\) such that \(f(x) = s(x) t(x)\text{.}\) Since \(\deg f = 2\) or 3, one of \(s(x)\) or \(t(x)\) has degree 1, and is thus of the form \(x-\alpha\text{,}\) where \(\alpha \in F\text{.}\) Thus, \(f\) has a zero.
    üîó
    Similarly, if \(f\) has a zero \(\alpha\in F\text{,}\) \(f(x) = (x-\alpha)g(x)\text{,}\) where \(\deg g \ge 1\text{.}\) Thus, \(f\) is reducible over \(F\text{.}\)
    üîó
    üîó

    As the following theorem illustrates, in \(F[x]\text{,}\) all irreducibles are primes.
    üîó

    Theorem 4.1.10.

    Let \(F\) be a field and \(p(x),f(x),g(x)\in F[x]\) such that \(p(x)\) is irreducible and \(p(x)\) divides \(f(x) g(x)\text{.}\) Then \(p(x)\) divides \(f(x)\) or \(p(x)\) divides \(g(x)\text{.}\)
    üîó
    üîó

    Proof.

    Assume \(p(x)\) does not divede \(f(x)\text{.}\) Then \(\gcd(p(x),f(x)=1\) and \(1=s(x)p(x)+t(x)f(x)\text{.}\) Multiplying by \(g(x)\) yields \(g(x)=g(x)s(x)p(x)+g(x)t(x)f(x)\) which implies that \(p(x)\) divides \(g(x)\) (since \(p(x)\) divides \(f(x)g(x)\)) .
    üîó
    üîó
    We next state the Fundamental Theorem of Algebra. Despite its name, its proof relies on analytic properties of the real numbers; there is no purely algebraic proof. Moreover, it is not essential for the work we do in following sections, but given its close relationship to the question of factorization, we include it here for completeness.
    üîó

    Fundamental Theorem of Algebra.

    Every nonconstant polynomial with coefficients in \(\C\) has a root in \(\C\text{.}\)
    üîó
    üîó
    We conclude with one consequence of the Fundamental Theorem of Algebra.

    üîó
    Thus, the multiplicative structure of \(\C[x]\) is straightforward: everything can be factored as a product of linear polynomials. Fields of coefficients like \(\C\) for which this is true are said to be algebraically closed; not all fields satisfy this property. For instance, \(x^2 + 1\in \R[x]\) does not factor into a product of linear polynomials. Consequently, \(\R\) is not algebraically closed.
    üîó
    However, regardless of whether our field is algebraically closed, we have not yet determined that any \(p\in F[x]\) can be factored uniquely into a product of irreducibles, or even that such factorizations into irreducibles exist.
