    Section 2.2 Rings

    Objectives: Guiding Questions

    In this section, we’ll seek to answer the questions:
    • What are rings and how do they relate to fields?
    • What are subrings, and how can we tell if a given subset of a ring is a subring?
    In the previous section, we observed that many familiar number systems are fields but that some are not. As we will see, these non-fields are often more structurally interesting, at least from the perspective of factorization. In this section, we explore them in more detail. We start that exploration with a rigorous definition of polynomials and their arithmetic operations.

    Subsection 2.2.1 Integral Polynomials

    Definition 2.2.1.

    A polynomial in \(x\) with coefficients in \(\Z\) is an expression of the form
    \begin{equation*} p(x) = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n\text{,} \end{equation*}
    where \(a_0,a_1,a_1,\ldots,a_n\in\Z\) and \(a_n\ne 0\text{.}\) We will sometimes omit the function notation and denote \(p(x)\) simply by \(p\text{.}\) We call \(n\in \N_0\) the degree of the polynomial \(p\text{,}\) denoted \(\deg(p) = n\text{,}\) and \(a_0,a_1,\ldots, a_n\) the coefficients of the polynomial. The coefficient \(a_n\) is known as the leading coefficient of \(p\text{,}\) and \(a_n x^n\) is the leading term of \(p\text{.}\) The set of all integral polynomials is:
    \begin{equation*} \Z[x] := \{a_0 + a_1 x + a_2 x + \cdots + a_n x^n\mid n\in \N_0, \ a_i\in \mathbb{Z}\}. \end{equation*}
    We know from our experience with polynomials in calculus and other courses that when we add two polynomials we again get a polynomial. Also when we multiply two polynomials we again get a polynomial. However, in order to show that addition and multiplication are well-defined operations on \(\Z[x]\text{,}\) they must first be defined!
    This is easier said than done. Let’s remind ourselves with an example that addition of polynomials is combine like terms. For example:
    \begin{equation*} (2x^2+(-3)x + 1) + (3x^3 + 2x^2 + x) = 3x^3 + 4x^2 + (-2)x+1. \end{equation*}
    Let’s do this computation slightly differently:
    \begin{align*} \amp (2x^2+(-3)x + 1) + (3x^3 + 2x^2 + x)\\ =\amp (0x^3+2x^2+(-3)x + 1) + (3x^3 + 2x^2 + x+0)\\ =\amp (0+3)x^3+(2+2)x^2+(-3+1)x + (1+0)\\ =\amp 3x^3+4x^2+(-2)x + 1. \end{align*}
    Notice, that to properly combine like terms we need to artificially add 0’s. This is a mental trick we tacitly do when we compute anyways but this idea of extending a polynomial by 0 is crucial to defining the addition and multiplication operations.

    Alternatives to Extension by 0.

    An alternative (more rigorous) way of defining addition and multiplication is to not use the variable \(x\) at all. But rather to think of polynomials as defined by their coefficients. In this case, we define a polynomial as a list of integers (indexed by \(\N_0\)): \((a_0,a_1,a_2,\ldots)\) such that only finitely many of the entries are non-zero. Or equivalently, that there exists \(N\in\N_0\) such that \(a_n=0\) for all \(n\ge N\text{.}\)

    Definition 2.2.2.

    Given two polynomials \(p(x),q(x)\in\Z[x]\text{,}\) we define their sum \(p(x)+q(x)\in\Z[x]\) as follows. Write
    \begin{align*} p(x)\amp= a_0+a_1x+a_2x^2+\cdots + a_nx^n\\ q(x)\amp= b_0+b_1x+b_2x^2+\cdots + b_mx^m. \end{align*}
    If \(n\geq m\text{,}\) we extend \(q(x)\) by 0:
    \begin{equation*} q(x) = b_0+b_1x+b_2x^2+\cdots + b_mx^m + b_{m+1}x^{m+1}+\cdots + b_nx^n \end{equation*}
    where \(b_{m+1}=\cdots = b_n = 0\text{.}\) If \(m\geq n\text{,}\) we extend \(p(x)\) by 0:
    \begin{equation*} p(x) = a_0+a_1x+a_2x^2+\cdots + a_nx^n + a_{n+1}x^{n+1}+\cdots + a_mx^m, \end{equation*}
    where \(a_{n+1} = \cdots = a_m = 0\text{.}\) Then
    \begin{equation*} p(x)+q(x) = (a_0+b_0) + (a_1+b_1)x + (a_2+b_2)x^2 + \cdots + (a_n+b_n)x^n. \end{equation*}
    The extra effort we’ve put in to defining polynomial addition will simplify proofs as the following proposition shows.

    Proposition 2.2.3.

    Addition of integral polynomials is commutative. That is, for any \(p(x),q(x)\in\Z[x]\text{,}\) we have
    \begin{equation*} p(x)+q(x) = q(x)+p(x). \end{equation*}

    Proof.

    Suppose \(p(x),q(x)\in\Z[x]\text{.}\) Possibly by extending by 0, we can assume
    \begin{align*} p(x)\amp= a_0+a_1x+a_2x^2+\cdots + a_nx^n\\ q(x)\amp= b_0+b_1x+b_2x^2+\cdots + b_nx^n. \end{align*}
    Then
    \begin{align*} p(x)+q(x)\amp= (a_0+a_1x+a_2x^2+\cdots + a_nx^n) + (b_0+b_1x+b_2x^2+\cdots + b_nx^n)\\ \amp= (a_0+b_0)+(a_1+b_1)x+(a_2+b_2)x^2+\cdots + (a_n+b_n)x^n \\ \amp= (b_0+a_0)+(b_1+a_1)x+(b_2+a_2)x^2+\cdots + (b_n+a_n)x^n \\ \amp= (b_0+b_1x+b_2x^2+\cdots + b_nx^n) + (a_0+a_1x+a_2x^2+\cdots + a_nx^n)\\ \amp = q(x)+p(x) \end{align*}
    Notice that in the proof of Proposition 2.2.3 we did not need to go into cases based off of if the degree of \(p(x)\) is greater than or less than the degree of \(q(x)\text{.}\)

    As a result of the answer to Exploration 2.2.3 and the completed Table 2.2.4, we make the following definition.

    Definition 2.2.5.

    A ring \(R\) is a nonempty set, together with binary operations \(+\) and \(\cdot\text{,}\) denoted \((R,+,\cdot)\text{,}\) and satisfying the following axioms.
    1. Given any \(a,b,c\in R\text{,}\) \((a+b)+c = a+(b+c)\text{.}\) (Associativity of addition)
    2. Given any \(a,b\in R\text{,}\) \(a+b= b+a\text{.}\) (Commutativity of addition)
    3. There exists an element \(0_R\in R\) such that for all \(a\in R\text{,}\) \(a+0_R = 0_R + a = a\text{.}\) (Additive identity)
    4. Given any \(a\in R\) there exists a \(b\in R\) such that \(a+b = b + a =0_R\text{.}\) (Additive inverses)
    5. Given any \(a,b,c\in R\text{,}\) \((a\cdot b)\cdot c = a\cdot (b\cdot c)\text{.}\) (Associativity of multiplication)
    6. There exists an element \(1_R\in R\text{,}\) such that \(a\cdot 1_R = 1_R\cdot a=a\) for all \(a\in R\text{.}\) (Multiplicative identity)
    7. For all \(a,b,c\in R\text{,}\) \(a\cdot (b+c) = a\cdot b + a\cdot c\text{.}\) (Left-distributive property)
    8. For all \(a,b,c\in R\text{,}\) \((a+b)\cdot c = a\cdot c + b\cdot c\text{.}\) (Right-distributive property)
    As with fields, when the ring \(R\) is clear from context, we will often write \(0\) in place of \(0_R\) and \(1\) in place of \(1_R\text{.}\) The element \(1_R\) is also referred to as the unity.

    Non-Unital Rings.

    Most rings of interest in modern algebra, particularly commutative algebra, have a multiplicative identity. Algebraic structures that satisfy all the ring axioms except for the multiplicative identity are called non-unital rings and are sometimes referred to as rngs (without the i).

    Subsection 2.2.3 Subrings: New Rings from Old

    Definition 2.2.9.

    Let \((R,+,\cdot)\) be a ring and let \(S\subseteq R\text{.}\) If \(S\) is itself a ring under \(+\) and \(\cdot\text{,}\) we say \(S\) is a subring of \(R\text{.}\) In this case, \(R\) is often called an overring of \(S\text{.}\)
    The following theorem provides a easy-to-apply test to check if a given subset \(S\) of a ring \(R\) is in fact a subring of \(R\text{.}\)

    Theorem 2.2.10.

    Let \(R\) be a ring and \(S\) a subset of \(R\text{.}\) Then \(S\) is a subring if and only if:
    1. \(1_R\in S\text{;}\)
    2. \(S\) is closed under multiplication; and
    3. \(S\) is closed under subtraction.

    Proof.

    Suppose \(1_R\in S\) and \(S\) is closed under multiplication and subtraction. We check each of the axioms in Definition 2.2.5. Since \(1_R\in S\) and \(S\) is closed under subtraction we have \(0_R = 1_R-1_R\in S\text{.}\) For any \(a\in S\text{,}\) we know that \(a\in R\) so that
    \begin{equation*} a+0_R = 0_R+a=a\text{ and }a\cdot 1_R = 1_R\cdot a = a. \end{equation*}
    Thus \(0_R\) and \(1_R\) are the additive and multiplicative identities, respectively, in \(S\text{.}\)
    For any \(b\in S\text{,}\) we know that \(-b=0_R-b\in S\text{.}\) Thus \(S\) is closed under additive inverses. If \(a,b\in S\text{,}\) then \(a+b=a-(-b)\in S\) so that \(S\) is closed under addition.
    For any triple of elements \(a,b,c\in S\text{,}\) we know \(a,b,c\in R\) so that \(a+(b+c) = (a+b)+c\) and \(a+b=b+a\text{.}\) Thus associativity and commutativity of addition are inherited from \(R\text{.}\) Similarly, \(a\cdot(b\cdot c) = (a\cdot b)\cdot c\text{,}\) \(a\cdot(b+c) = a\cdot b + a\cdot c\text{,}\) and \((a+b)\cdot c = a\cdot c + b\cdot c\text{.}\) That is, \(S\) inherits associativity of multiplication as well as the left- and right-distributive properties from \(R\text{.}\)
    We conclude that \(S\) is a ring under the multiplication and addition from \(R\text{.}\)

    Subsection 2.2.4 Units and Associates in a Ring

    Definition 2.2.11.

    Let \(R\) be a ring and let \(u\in R\) be nonzero. If there is a \(v\in R\) such that \(uv = vu = 1\text{,}\) we say \(u\) is unit of \(R\text{.}\) We denote the set of units of \(R\) by \(R^\times\text{.}\) We say \(x,y\in R\) are associates if there exists some \(u\in R^\times\) such that \(x = uy\text{.}\)

    Exploration 2.2.9.

    Explicitly describe the set \(\Z^\times\text{.}\) What are the associates of 7 in \(\Z\text{?}\)
    In other words, a unit in a ring is a nonzero element with a multiplicative inverse. The existence of units is the primary difference between fields and commutative rings with identity: in a field, all nonzero elements are units, while in a commutative ring with identity, no nonzero elements need be units, as Theorem 2.2.12 demonstrates.

    Theorem 2.2.12.

    A commutative ring with identity \(R\) in which every nonzero element is a unit is a field.

    Proof.

    Compare the axioms for a commutative ring with identity and a field. The only thing missing from the ring axioms is the existence of multiplicative inverses for nonzero elements.
