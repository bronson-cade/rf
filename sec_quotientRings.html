    Section 3.3 Quotient Rings

    Guiding Questions: Guiding Questions

    In this section, we’ll seek to answer the questions:
    • How can we use ideals to build new rings out of old?
    • What sorts of ideals allow us to build domains? Fields?
    • What is the First Isomorphism Theorem?
    If the only rings that existed were polynomial rings, familiar systems of numbers like \(\Z/m, \Z, \Q, \R, \C\text{,}\) polynomial rings, and matrix rings, there would still be enough to justify defining the concept of a ring and exploring its properties. However, these are not the only rings that exist. In this section, we explore a way of building new rings from old by means of ideals. To better understand these new rings, we will also define two new classes of ideals: prime ideals, and maximal ideals. We end by briefly connecting these rings to a familiar problem from high school algebra.

    Subsection 3.3.1 Congruence modulo \(I\)

    The major concept of this section is the notion of congruence modulo \(I\text{.}\) One can reasonably think of this idea as a generalization of congruence modulo \(m\) in \(\Z\text{.}\)

    Definition 3.3.1.

    Let \(R\) be a ring, \(I\) an ideal of \(R\text{,}\) and \(a,b\in R\text{.}\) We say \(a\) is congruent to \(b\) modulo \(I\) if \(b-a\in I\text{.}\) If this is the case, we write \(a + I = b + I\text{.}\)

    Activity 3.3.1.

    Determine (with brief justification) whether \(a + I = b + I\) in the following rings \(R\text{.}\)
    1. \(a = 9\text{,}\) \(b = 3\text{,}\) \(I = (6)\text{,}\) \(R = \Z\)
    2. \(a = 10\text{,}\) \(b = 4\text{,}\) \(I = (7)\text{,}\) \(R = \Z\)
    3. \(a = 9\text{,}\) \(b = 3\text{,}\) \(I = (6)\text{,}\) \(R = \Z[x]\)
    4. \(a = x^2+x-2\text{,}\) \(b = x-1\text{,}\) \(I = (x+1)\text{,}\) \(R = \Q[x]\)
    5. (Challenge.) \(a = x^3\text{,}\) \(b = x^2+2x\text{,}\) \(I = (y-x^2, y-x-2)\text{,}\) \(R = \Q[x,y]\)
    Solution.
    1. We have \(3-9 = -6 = (-1)\cdot 6\in (6)\) so \(9+(6) = 3+ (6)\) in \(\Z\text{.}\)
    2. We have \(4-10 = -6\notin (7)\) as \(-6\neq 7k\) for any \(k\in\Z\text{.}\) Thus \(10+(7)\neq 4+(7)\) in \(\Z\text{.}\)
    3. \(3-9 = -6 = (-1)\cdot 6\in (6)\text{.}\) Thus \(9+(6) = 3+(6)\) in \(\Z[x]\text{.}\)
    4. \(x-1-(x^2+x-2) = -x^2+1 = -(x-1)(x+1)\in (x+1)\text{.}\) Thus \(x-1+(x+1) = x^2+x-2 + (x+1)\) in \(\Q[x]\text{.}\)
    5. \(x^3 - x^2 - 2x = x(x-2)(x+1) = -x(y-x^2) + x(y-x-2)\text{.}\) Thus
      \begin{equation*} x^3+(y-x^2,y-x-2) = x^2+x+ (y-x^2,y-x-2) \end{equation*}
      in \(\Q[x,y]\text{.}\)

    Exploration 3.3.2.

    Given a ring \(R\text{,}\) ideal \(I\text{,}\) and \(a\in R\text{,}\) when is it the case that \(a + I = 0 + I = I\text{?}\)
    Answer.
    When \(a\in I\text{.}\)
    Observe that if \(b-a \in I\text{,}\) then there is some \(x\in I\) such that \(b-a = x\text{,}\) and so \(b = a+x\text{.}\)
    The notation \(a+I = b+I\) is more than just notation. It is an equality of sets. Given any ideal \(I\subseteq R\) and any \(a\in R\text{,}\) define
    \begin{equation*} a+I = \{a+i\mid i\in I\} \end{equation*}
    to be the set of all translates of \(I\) under \(a\text{.}\)

    Theorem 3.3.2.

    Let \(I\subseteq R\) be an ideal and \(a,b\in R\text{.}\) Then \(a\) is congruent to \(b\) modulo \(I\) if and only if we have equality of sets \(a+I = b+I\text{.}\)

    Proof.

    Suppose \(a\) is congruent to \(b\) modulo \(I\text{.}\) Then \(b-a\in I\) so that \(b = a+x\) for some \(x\in I\text{.}\) Take any \(i\in I\text{,}\) then \(b+i = a+(x+i)\in a+I\) as \(x+i\in I\text{.}\) Thus \(b+I\subseteq a+I\text{.}\) Using instead \(a = b+(-x)\) we have, similarly, that \(a+I\subseteq b+I\text{.}\) Thus \(a+I = b+I\text{.}\)
    Suppose \(a+I = b+I\text{.}\) Since \(0\in I\text{,}\) we know that \(b = b+0\in b+I\text{.}\) Since \(a+I = b+I\text{,}\) we have that \(b\in a+I\text{.}\) Thus \(b = a + x\) for some \(x\in I\text{.}\) Equivalently \(b-a=x\in I\text{.}\) Thus \(a\) is congruent to \(b\) modulo \(I\text{.}\)

    Definition 3.3.3.

    Given an ideal \(I\subseteq R\) in a commutative ring \(R\text{.}\) A coset of \(I\) is a set of the form
    \begin{equation*} a+I = \{a+i\mid i\in I\}. \end{equation*}
    The collection of all cosets is denoted \(R/I\) or \(\bar{R}\) if \(I\) is understood:
    \begin{equation*} R/I = \{a+I\mid a\in R\}. \end{equation*}

    Example 3.3.4.

    We take a minute to observe that in \(\Z\) the notion of congruence: \(a\equiv b\) modulo \(m\) is equivalent to the definition here. Indeed, any \(m\in\Z\) defines a principal ideal \((m)\text{.}\) If \(a\equiv_m b\text{,}\) then \(b-a = mk\) for some \(k\in \Z\text{.}\) Hence, \(b-a\in (m)\text{.}\) And conversely, if \(a+(m) = b+(m)\text{,}\) then \(b-a\in (m)\text{.}\) Thus there exists \(k\in\Z\) with \(b-a = mk\) and so \(a\equiv_mb\text{.}\)
    In particular, this is a successful generalization of the notion of congruence modulo \(m\text{.}\)
    As was the case in \(\Z_m\text{,}\) congruence modulo \(I\) is an equivalence relation.

    Theorem 3.3.5.

    Let \(R\) be a ring and \(I\) an ideal of \(R\text{.}\) Then congruence modulo \(I\) is an equivalence relation on \(R\text{.}\)

    Proof.

    Since \(0\in I\text{,}\) \(a-a \in I\) for all \(a\text{,}\) so \(a + I = a+I\) and the relation is reflexive.
    Moreover, if \(b-a \in I\text{,}\) then \(-1(b-a) = a-b \in I\text{,}\) so the relation is symmetric.
    If \(b-a \in I\) and \(c-b\in I\text{,}\) then \((b-a) + (c-b) = c-a \in I\text{,}\) so the relation is transitive.

    Subsection 3.3.2 Quotient Rings

    Notation for Integers Modulo \(m\).

    It’s standard practice to identify the principal ideal \((r)\) with \(r\) when no confusion can arise. In \(\Z\text{,}\) the set of equivalence classes modulo \((m)\) is denoted by \(\Z/m\) instead of \(\Z/(m)\text{.}\)
    The set of equivalence classes under this relation is denoted \(R/I\text{.}\) In particular, What is more, this is not merely a set of equivalence classes. As the next two theorems demonstrate, this set possesses two algebraic operations that extend naturally from those of \(R\text{.}\)

    Theorem 3.3.6.

    Let \(R\) be a ring and \(I\) an ideal of \(R\text{.}\) If \(a_1,a_2,b_1,b_2\in R\) such that \(a_1+I = a_2+I\) and \(b_1+I = b_2+I\text{,}\) then \((a_1+b_1) + I = (a_2+b_2) + I\text{.}\)

    Proof.

    Suppose \(a_1+I = a_2+I\) and \(b_1+I = b_2+I\text{.}\) Then \(a_2-a_1 = x \in I\) and \(b_2-b_1 = y \in I\text{.}\) Adding, we have
    \begin{equation*} (a_2+b_2) - (a_1+b_1) =(a_2-a_1) + (b_2-b_1) = x+y \in I. \end{equation*}
    Thus, \((a_1+b_1) + I = (a_2+b_2) + I\text{.}\)
    Note that the only requirement in the proof of this theorem is that \(I\) is closed under addition.

    Theorem 3.3.7.

    Let \(R\) be a commutative ring and \(I\) an ideal of \(R\text{.}\) If \(a_1,a_2,b_1,b_2\in R\) such that \(a_1+I = a_2+I\) and \(b_1+I = b_2+I\text{,}\) then \(a_1b_1 + I = a_2b_2 + I\text{.}\)

    Proof.

    As before, we observe that \(a_2-a_1 = x\in I\) and \(b_2-b_1 = y\in I\text{.}\) Write \(a_2 = a_1+x\) and \(b_2 = b_1+y\text{.}\) Then
    \begin{align*} a_2b_2 =\amp (a_1+x)(b_1+y) \\ \amp= a_1b_1+a_1y +xb_1+xy\\ \amp= a_1b_1 + \underbrace{(a_1y+xb_1+xy)}_{\in I}\text{,} \end{align*}
    so \(a_2b_2-a_1b_1 = a_1y+xb_1+xy \in I\text{.}\) Therefore \(a_1b_1 + I = a_1b_1 + I\text{.}\)
    The preceeding proof used two properties of \(I\text{:}\) closure under addition and closure under multiplication by \(R\text{.}\)
    The previous two theorems together show that addition and multiplication on the set \(R/I\) is well-defined. As these operations are built on the operations of \(R\text{,}\) it will likely not surprise you to learn that the usual axioms defining a ring also hold.

    Theorem 3.3.8.

    Let \(R\) be a commutative ring, \(I\) an ideal of \(R\text{,}\) and \(a,b\in R\text{.}\) The set of cosets of \(I\text{,}\) denoted \(\bar{R} = R/I\text{,}\) is a commutative ring (called the quotient ring of \(R\) by \(I\)) with additive identity \(0_R+I\) and multiplicative identity \(1_R + I\) under the following operations:
    \begin{align*} (a+I) + (b+I) & := (a+b) + I\\ (a+I) \cdot (b+I) & := (ab) + I. \end{align*}

    Proof.

    These operations were shown to be well-defined in Theorem 3.3.6 and Theorem 3.3.7. We need to show that, with these operations, all eight ring axioms hold and that multiplication is commutative. Take \(a+I,b+I,c+I\in R/I\text{.}\)
    We begin with the additive axioms. For associativity:
    \begin{align*} (a+I) + ((b+I)+(c+I)) \amp= (a+I) + ((b+c)+I) \\ \amp= (a+(b+c))+I\\ ((a+I) + (b+I))+(c+I) \amp= ((a+b)+I) + (c+I) \\ \amp= ((a+b)+c)+I. \end{align*}
    Since \(R\) has associative addition we have \(a+(b+c) = (a+b)+c\) and so
    \begin{equation*} a+(b+c) +I = (a+b)+c + I \end{equation*}
    the cosets are equal.
    For commutativity:
    \begin{equation*} (a+I) + (b+I) = (a+b) + I = (b+a)+I = (b+I)+(a+I). \end{equation*}
    For identity:
    \begin{equation*} (0_R+I)+(a+I) = (0_R+a) + I = a+I \end{equation*}
    as \(0_R\) is the additive identity in \(R\text{.}\)
    For inverses:
    \begin{equation*} (a+I) + ((-a)+I) = (a+(-a)) + I = 0_R + I \end{equation*}
    and so the additive inverse of \(a+I\) is \(-a+I\text{.}\)
    We now turn to the multiplicative axioms plus commutativity. For associativity:
    \begin{align*} (a+I)((b+I)(c+I))\amp= (a+I)(bc+I) \\ \amp= a(bc) + I \\ ((a+I)(b+I))(c+I) \amp= (ab+I)(c+I) \\ \amp= (ab)c+I \end{align*}
    But multiplication in \(R\) is associative so that \(a(bc) = (ab)c\text{.}\) Hence,
    \begin{equation*} a(bc)+I = (ab)c+I. \end{equation*}
    For commutativity:
    \begin{equation*} (a+I)(b+I) = (ab)+I = (ba)+I = (b+I)(a+I). \end{equation*}
    For identity:
    \begin{equation*} (1_R+I)(a+I) = (1_R\cdot a)+I = a+I \end{equation*}
    as \(1_R\) is the multiplicative identity in \(R\text{.}\)
    Since \(R/I\) has commutative multiplication we only need to check the left-distributive axiom.
    \begin{align*} (a+I)( (b+I) + (c+I)) \amp= (a+I)( (b+c) + I) \\ \amp= (a(b+c)) + I \\ \amp= (ab+ac) + I\\ (a+I)(b+I)+ (a+I)(c+I) \amp= (ab)+I + (ac)+I \\ \amp= (ab+ac) + I \end{align*}
    where we have used that multiplication in \(R\) has the left-distributive property in line 3.
    We conclude that \(R/I\) is a commutative ring.
    Thus, given a ring \(R\) and ideal \(I\) of \(R\text{,}\) we may build the quotient ring \(R/I\text{.}\)

    Subsection 3.3.3 Prime and Maximal Ideals

    In this section, we continue our exploration of quotient rings by looking more closely at properties of ideals. We focus on particular properties of ideals that ensure that the quotient \(R/I\) is either a domain or a field.

    Fraktur Font.

    The Fraktur font is a German typeface from the 16th century. It was long known as the “German typeface”. In 1941 it was banned by the Nazi government. It is usually used when denoting prime (\(\mathfrak{p}\)) and maximal (\(\mathfrak{m}\)) ideals. TBD

    Definition 3.3.9.

    Let \(R\) be commutative with identity and \(\mathfrak{p}\subsetneq R\) a non-zero proper ideal. We say \(\mathfrak{p}\) is prime if whenever \(a,b\in R\) such that \(ab\in \mathfrak{p}\text{,}\) we have \(a\in \mathfrak{p}\) or \(b\in \mathfrak{p}\text{.}\)
    This definition arises by replacing divides with is contained in in the usual definition of prime element??? The following theorem shows that these definitions agree to the extent they can: prime elements generate prime ideals.

    Theorem 3.3.10.

    Let \(R\) be a domain and \(p\in R\) be prime. Then \(\mathfrak{p} = (p)\) is a prime ideal.

    Proof.

    Suppose \(p\) is a prime element of a domain and \(\mathfrak{p} = (p)\text{.}\) Suppose a product is in \(\mathfrak{p}\text{,}\) say \(ab\in\mathfrak{p}\text{.}\) Then there exists \(r\in R\) with \(rp = ab\text{.}\) Thus \(p\mid ab\text{.}\) Since \(p\) is prime, either \(p\mid a\) or \(p\mid b\text{.}\)
    Without loss of generality assume \(p\mid a\text{.}\) Then there exists \(s\in R\) with \(a = sp\text{.}\) Hence, \(a\in (p) = \mathfrak{p}\text{.}\)

    It is this precise condition that guarantees that the resulting quotient is a domain.

    Theorem 3.3.11.

    Let \(R\) be commutative with identity and \(I\) an ideal of \(R\text{.}\) Then \(I\) is prime if and only if \(R/I\) is an integral domain.

    Proof.

    Begin by assuming that \(I\) is prime, and suppose \((a+I)(b+I) = 0+I = I\text{.}\) Then \(ab \in I\text{,}\) and since \(I\) is prime, either \(a\in I\) or \(b\in I\text{.}\) Thus either \(a+I = 0+I\) or \(b+I = 0+I\text{.}\)
    Now assume that \(R/I\) is a domain. Further, let \(a,b\in R\) be such that \((a+I)(b+I) = ab+I = 0+I\text{.}\) Then \(ab\in I\text{,}\) and since \(R/I\) is a domain, \(a+I = 0+I\) or \(b+I = 0+I\text{,}\) i.e., \(a\in I\) or \(b\in I\text{.}\) Thus, \(I\) is prime.
    We now consider another important class of ideals: the maximal ideals.

    Definition 3.3.12.

    Let \(R\) be commutative with identity and let \(\mathfrak{m}\subsetneq R\) be a non-zero proper ideal. We say that \(\mathfrak{m}\) is a maximal ideal if no proper ideal of \(R\) properly contains \(M\text{.}\) That is, if \(J\) is an ideal satisfying \(\mathfrak{m}\subseteq J\subseteq R\text{,}\) either \(J=\mathfrak{m}\) or \(J=R\text{.}\)
    In other words, an ideal \(\mathfrak{m}\ne R\) is maximal if no “larger” ideal (with respect to inclusion) properly contains it. As we will see later, rings can have many maximal ideals.
    It is a fact that any ring \(R\) with \(0_R\ne 1_R\) has a maximal ideal. This follows from Zorn’s Lemma; a rigorous exploration of Zorn’s Lemma lies outside of the scope of this text, but suffice it to say that Zorn’s Lemma is incredibly useful in all areas of algebra for proving existence theorems. For example, a proof that every vector space has a basis relies on Zorn’s Lemma.
    Rings with only one maximal ideal are said to be local rings, and are actively studied in modern research in commutative algebra (the study of commutative rings and their properties).
    The next two results demonstrate that the maximality of \(I\) is precisely the condition that guarantees that \(R/I\) is a field.

    For the forward direction, apply the previous lemma to construct an inverse for \(x+I\) given any \(x\in R\setminus I\text{.}\)

    Proof.

    If \(R/I\) is a field, assume \(J\) is an ideal of \(R\) that properly contains \(I\text{.}\) Let \(x\in J\setminus I\text{;}\) then \(x+I\) is a nonzero element of \(R/I\text{,}\) and since \(R/I\) is a field, there is some \(y+I\) such that \((xy)+I = 1+I\text{.}\) Since \(x\in J\text{,}\) \(xy\in J\text{.}\) As \(1+I = (x+I)(y+I) = xy+I\text{,}\) we have \(1-xy\in I\subsetneq J\text{,}\) and thus \(1 = (1-xy)+xy \in J\text{,}\) which means \(J = R\text{.}\) Thus, \(I\) is maximal.
    Now, suppose that \(I\) is maximal and let \(x\in R\setminus I\text{.}\) Apply the previous lemma to obtain \(1 = xr' + y'\text{,}\) where \(y'\in I\text{.}\) Then
    \begin{equation*} 1 + I = xs+y + I = xs + I = (x+I)(y+I). \end{equation*}

    Theorem 3.3.15.

    Every maximal ideal is prime.

    Proof.

    All fields are integral domains. Thus, if \(I\) is maximal, \(R/I\) is a field, thus a domain, and thus \(I\) is prime.
    In general, the converse is not true (see the Challenge below). However, it holds in sufficiently nice rings.

    Theorem 3.3.16.

    In a principal ideal domain, every prime ideal is maximal.

    Proof.

    Let \(R\) be a PID and \(\mathfrak{p}=(p)\) a prime ideal. By previous work, \(p\) is prime. Suppose that \(\mathfrak{p} \subseteq \mathfrak{m}\text{.}\) Thus, \(p\in \ideal{m}\text{,}\) so \(m|p\text{.}\) That is, \(p = mk\text{.}\) Since \(p\) is prime and \(R\) is a domain, it is irreducible. Thus, either \(m\) or \(k\) is a unit. If \(m\) is a unit, then \(\ideal{m} = R\text{.}\) If \(k\) is a unit, then \(m = k^{-1} p\text{,}\) and thus \(m\in \ideal{p}\text{,}\) which means that \(\ideal{m} = \ideal{p}\text{.}\)
    Thus, \(\ideal{p}\) is maximal.

    Subsection 3.3.4 Homomorphisms and Quotient Rings

    As quotient rings provide fertile soil for building new examples of rings, it should not surprise us to find that homomorphisms interact with quotient rings in interesting and useful ways. Chief among them are the isomorphism theorems. In this subsection, we focus primarily on the First Isomorphism Theorem.
    We have seen that any homomorphism \(\p : R\to S\) gives rise to an ideal of \(R\text{,}\) namely \(\ker\p\text{.}\) Our next theorem demonstrates that, given a commutative ring with identity \(R\text{,}\) every ideal is the kernel of some homomorphism defined on \(R\text{.}\)

    Theorem 3.3.17.

    Let \(R\) be commutative with identity and \(I\) an ideal of \(R\text{.}\) Define \(\p: R\to R/I\) by \(\p(r) = r+I\text{.}\) Then \(\p\) is a homomorphism with \(\ker\p = I\text{.}\)
    In what follows, we work toward a proof of the First Isomorphism Theorem for Rings.
    Throughout, let \(R\) and \(S\) be commutative rings with identity, and let \(\p : R\to S\) be a homomorphism. Recall that \(\im \p = \{s\in S\mid \p(r) = s\text{ for some } r\in R}\text{.}\)
    Define \(f: R/\ker \p \to \im \p\) by \(f(r+\ker \p) = \p(r)\text{.}\)

    Lemma 3.3.18.

    Using the notation from above, \(f\) is a well-defined function.

    Proof.

    Suppose that \(r_1 + \ker\p = r_2 + \ker\p\text{.}\) Then \(r_2-r_1 \in \ker\p\text{,}\) so \(\p(r_2-r_1) = 0_S\text{,}\) and thus \(\p(r_1) = \p(r_2)\text{.}\) Therefore, \(f(r_1+\ker\p) = f(r_2+\ker\p)\text{,}\) and \(f\) is well-defined.

    Lemma 3.3.19.

    Using the notation above, \(f\) is a homomorphism.

    Proof.

    We show that \(f\) preserves addition. That it preserves multiplication will follow similarly. Observe that \(f((x+\ker\p) + (y+\ker\p)) = f((x+y)+\ker\p) = \p(x+y) = \p(x)+\p(y) = f(x+\ker\p) + f(y+\ker\p)\text{.}\)

    Lemma 3.3.20.

    Using the notation above, \(f\) is one-to-one.

    Proof.

    Suppose that \(f(r_1 + \ker\p) = f(r_2 + \ker\p)\text{.}\) That is, \(\p(r_1) = \p(r_2)\text{.}\) Then \(\p(r_1 - r_2) = 0_S\text{,}\) so \(r_1 -r_2 \in \ker\p\text{,}\) and therefore \(r_1 + \ker\p = r_2 + \ker \p\text{.}\) Thus, \(f\) is one-to-one.

    Lemma 3.3.21.

    Using the notation above, \(f\) is onto.

    Proof.

    Since \(\p\) is onto \(\im \p\) by definition, given any \(s\in \im \p\) there is some \(r\in R\) such that \(\p(r) = s\text{.}\) Then \(f(r+\ker\p) = \p(r)\text{.}\)
    We thus obtain the following, due to Emmy Noether:

    Theorem 3.3.22 (E. Noether). Noether’s Isomorphism Theorem.

    Let \(\p : R\to S\) be a homomorphism of commutative rings. Then \(R/\ker \p \cong \im \p\text{.}\)
    In particular, if \(\p : R\to S\) is onto, \(R/\ker \p \cong S\text{.}\)
    The First Isomorphism Theorem gives a useful way of establishing an isomorphism between a quotient ring \(R/I\) and another ring \(S\text{:}\) find an onto homomorphism \(R\to S\) with kernel \(I\text{.}\)
