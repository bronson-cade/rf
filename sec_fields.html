    Guiding Questions

    In this section, we‚Äôll seek to answer the questions:
    üîó
    • What are binary operations?
      üîó
    • What is a field? What sorts of things can one do in a field?
      üîó

      üîó

    Subsection 2.1.1 Number Systems

    We now begin the process of abstraction. We will do this in stages, beginning with the concept of a field. First, we need to formally define some familiar sets of numbers.
    üîó

    Definition 2.1.1.

    The rational numbers, denoted by \(\Q\text{,}\) is the set
    \begin{equation*} \Q = \left\{\frac{a}{b}\mid a,b\in \Z, \ b\ne 0\right\}\text{.} \end{equation*}
    The symbol is \(\frac{a}{b}\) is a fraction and we regard two symbols \(\frac{a}{b},\frac{c}{d}\in \Q\) as defining the same element if and only if \(ad = bc\text{.}\)
    üîó
    üîó

    Activity 2.1.1.

    Prove that the relation defined in Definition¬†2.1.1 is an equivalence relation. Recall Definition¬†1.4.2. We should regard the relation as being defined one the set
    \begin{equation*} S = \{(a,b)\mid a,b\in\mathbb{Z}, b\neq 0\} \end{equation*}
    where we regard the fraction \(\frac{a}{b}\) as the ordered pair \((a,b)\text{.}\)
    üîó
    üîó

    Proposition 2.1.2.

    For \(\frac{a}{b},\frac{c}{d}\in\Q\text{,}\) the sum and product:
    \begin{equation*} \frac{a}{b}+\frac{c}{d} = \frac{ad+bc}{bd}\text{ and }\frac{a}{b}\cdot \frac{c}{d} = \frac{ac}{bd} \end{equation*}
    is well-defined.
    üîó
    üîó

    Proof.

    We only show the product is well-defined, while the sum is left as an exercise. To be well-defined, we need to argue that for any equivalent ways of representing both fractions in the product, the result is unchanged. To that end, take \(\frac{a}{b},\frac{a'}{b'},\frac{c}{d},\frac{c'}{d'}\in\Q\text{.}\) Suppose that \(ab'=a'b\) and \(cd' = c'd\text{.}\)
    üîó
    Then
    \begin{align*} \frac{a}{b}\cdot \frac{c}{d}\amp= \frac{ac}{bd}\\ \frac{a'}{b'}\cdot \frac{c'}{d'}\amp= \frac{a'c'}{b'd'} \end{align*}
    and we need to show that \(\frac{ac}{bd}\) is equivalent to \(\frac{a'c'}{b'd'}\) or that \(acb'd' = a'c'bd\text{.}\)
    üîó
    We have
    \begin{align*} acb'd' \amp= (ab')(cd') \\ \amp= (a'b)(c'd) \\ \amp= a'c'bd \end{align*}
    as desired.
    üîó
    üîó
    We could attempt define any operations we like on \(\Q\text{.}\) It does not mean they are well-defined. To be well-defined we must actually get a function out of our definition. For example, we could define the rule \(\oplus\) by \(\frac{a}{b}\oplus \frac{c}{d} = \frac{1}{bd}\text{.}\) But this would not make sense at all. Indeed, we have \(\frac{2}{4} = \frac{1}{2}\) so it should then be true that \(\frac{2}{4}\oplus \frac{1}{1} = \frac{1}{2}\oplus\frac{1}{1}\) as \(\frac{2}{4}\) and \(\frac{1}{2}\) are equivalent fractions. Yet, we can compute \(\frac{2}{4} \oplus\frac{1}{1} = \frac{1}{4\cdot 1}\) and \(\frac{1}{2}\oplus \frac{1}{1} = \frac{1}{2\cdot 1}\text{.}\) Since \(\frac{1}{4}\neq \frac{1}{2}\text{,}\) the rule \(\oplus\) is not well-defined. This indicates some care must be taken when dealing with any operation or function defined on equivalence classes of numbers (or objects) as we have just done for addition and multiplication on \(\Q\text{.}\)
    üîó

    Activity 2.1.2.

    Write down two binary operations on \(\Q\) that are not well-defined.
    üîó
    üîó
    We likely have an intuitive idea of what is meant by \(\R\text{,}\) the set of real numbers. Defining \(\R\) rigorously is actually quite difficult, and occupies a significant amount of time in a first course in real analysis. Thus, we will make use of your intuition.
    üîó
    It is worth noting that \(\N\subseteq \Z\subseteq \Q\subseteq \R\text{,}\) but \(\R\) also contains many irrational numbers, such as \(\pi\text{,}\) \(e\text{,}\) \(\sqrt{2}\text{,}\) and so on.
    üîó
    Out of \(\R\) we may build the complex numbers.
    üîó

    Definition 2.1.3.

    The complex numbers consist of all expressions of the form \(a+bi\text{,}\) where \(a,b\in \R\) and \(i^2 = -1\text{.}\) Given \(z = a+bi\text{,}\) we say \(a\) is the real part of \(z\) and \(b\) is the imaginary part. The set of complex numbers is denoted \(\C\text{.}\)
    üîó
    Addition and multiplication in \(\C\) is defined in the usual way:
    \begin{align*} (a_1+b_1i)+(a_2+b_2i)\amp= (a_1+a_2)+(b_1+b_2)i \\ (a_1+b_1i)\cdot (a_2+b_2i)\amp= (a_1a_2-b_1b_2)+(a_1b_2+a_2b_1)i \end{align*}
    üîó
    üîó
    Unlike \(\Q\text{,}\) the elements of \(\C\) are not equivalence classes and so we do not need an analogue of Proposition¬†2.1.2 here.
    üîó
    üîó

    Subsection 2.1.2 Binary Operations

    Algebra comes from an Arabic word meaning ‚Äúthe reunion of broken parts‚Äù. We therefore need a way of combining two elements of a set into one; we turn to a particular type of function, known as a binary operation, to accomplish this.
    üîó

    Definition 2.1.4.

    Let \(X\) be a nonempty set. A function \(\star \colon X \times X \to X\) is called a binary operation. If \(\star\) is a binary operation on \(X\text{,}\) we say that \(X\) is closed under the operation \(\star\). [Given \(a,b\in X\text{,}\) we usually write \(a\star b\) in place of the typical function notation, \(\star(a,b)\text{.}\)]
    üîó
    üîó

      üîó
      üîó
    2. on \(\Q\text{?}\)
      üîó
      üîó
    3. on \(\Z\text{?}\)
      üîó
      üîó
    4. on \(\N\text{?}\)
      üîó
      üîó

    Subsection 2.1.3 Fields

    The hallmark of modern pure mathematics is the use of axioms. An axiom is essentially an unproved assertion of truth. Our use of axioms serves several purposes.
    üîó
    From a logical perspective, axioms help us avoid the problem of infinite regression (e.g., asking How do you know? over and over again). That is, axioms give us very clear starting points from which to make our deductions.
    üîó
    To that end, our first abstract algebraic structure captures and axiomatizes familiar behavior about how numbers can be combined to produce other numbers of the same type.
    üîó

    Aside: Heinrich Martin Weber.

    The first whispers of a field were known to Lagrange in 1770. However, it was not until 1893 that a formal definition of an abstract field was given. This definition is due to Heinrich Martin Weber in the paper Die allgemeinen Grundlagen der Galois‚Äôschen Gleichungstheorie. Weber was a German mathematician known for contributions to the fields of algebra, number theory, and analysis.
    üîó

    Definition 2.1.5.

    A field is a nonempty set \(F\) with at least two elements and binary operations \(+\) and \(\cdot\text{,}\) denoted \((F,+,\cdot)\text{,}\) and satisfying the following field axioms:
    1. Given any \(a,b,c\in F\text{,}\) \((a+b)+c = a+(b+c)\text{.}\) (Associativity of addition)
      üîó
      üîó
    2. Given any \(a,b\in F\text{,}\) \(a+b= b+a\text{.}\) (Commutativity of addition)
      üîó
      üîó
    3. There exists an element \(0_F\in F\) such that for all \(a\in F\text{,}\) \(a+0_F = 0_F + a = a\text{.}\) (Additive identity)
      üîó
      üîó
    4. Given any \(a\in F\) there exists a \(b\in F\) such that \(a+b = b + a =0_F\text{.}\) (Additive inverse)
      üîó
      üîó
    5. Given any \(a,b,c\in F\text{,}\) \((a\cdot b)\cdot c = a\cdot (b\cdot c)\text{.}\) (Associativity of multiplication)
      üîó
      üîó
    6. Given any \(a,b\in F\text{,}\) \(a\cdot b = b\cdot a\text{.}\) (Commutativity of multiplication)
      üîó
      üîó
    7. There exists an element \(1_F\in F\) such that for all \(a\in F\text{,}\) \(1_F\cdot a = a\cdot 1_F = a\text{.}\) (Multiplicative identity)
      üîó
      üîó
    8. For all \(a\in F\text{,}\) \(a\ne 0_F\text{,}\) there exists a \(b\in F\) such that \(a\cdot b = b\cdot a = 1_F\text{.}\) (Multiplicative inverse)
      üîó
      üîó
    9. For all \(a,b,c\in F\text{,}\) \(a\cdot (b+c) = a\cdot b + a\cdot c\text{.}\) (Distributive property I)
      üîó
      üîó
    10. For all \(a,b,c\in F\text{,}\) \((a+b)\cdot c = a\cdot c + b\cdot c\text{.}\) (Distributive property II)
      üîó
      üîó
    üîó
    üîó
    We will usually write \(a\cdot b\) as \(ab\text{.}\) Additionally, we will usually drop the subscripts on \(0,1\) unless we need to distinguish between fundamentally different identities in different fields.
    üîó

    Subsection 2.1.4 Examples of Fields

    We will now provide more rigorous arguments that the fields investigated are in fact fields. We will do so by checking, for a handful of the fields investigated, each of the axioms hold. At the end of the section we will introduce a way of simplyfying some of our work.
    üîó

    Example 2.1.6. Rational Numbers are a Field.

    Having already shown that addition and multiplication are well-defined in Proposition¬†2.1.2. We show that, equipped with these binary operations, \(\Q\) forms a field. Take \(\frac{a}{b}, \frac{c}{d}, \frac{e}{f}\in \Q\text{.}\)
    üîó
    We start with the additive checks: associativity, commutativity, additive identity, and additive inverses. For associativity we have:
    \begin{align*} \frac{a}{b} + \left(\frac{c}{d}+\frac{e}{f}\right) \amp= \frac{a}{b} + \left(\frac{cf+de}{cf}\right)\\ \amp= \frac{a(cf)+b(cf+de)}{b(cf)}\\ \amp= \frac{acf + bcf+bde}{bcf} \end{align*}
    where we have dropped the parentheses in the numerator as multiplication in \(\Z\) is associative. We also have
    \begin{align*} \left(\frac{a}{b} + \frac{c}{d}\right)+\frac{e}{f} \amp= \frac{ac+bd}{bd} + \frac{e}{f}\\ \amp= \frac{(ac+bd)f+bd(e)}{b(cf)}\\ \amp= \frac{acf+bcf+bde}{bcf} \end{align*}
    and this is the same as the other association.
    üîó
    For commutativity we have:
    \begin{align*} \frac{a}{b} +\frac{c}{d} \amp= \frac{ad+bc}{bd} \\ \frac{c}{d} +\frac{a}{b} \amp= \frac{cb+da}{db} \end{align*}
    But in \(\Z\) we know that \(db = bd\) and \(ad+bc=cb+da\text{.}\)
    üîó
    The element \(\frac{0}{1}\in\Q\) is the additive identity as
    \begin{equation*} \frac{0}{1}+\frac{a}{b} = \frac{0\cdot b + 1\cdot a}{1\cdot b} = \frac{a}{b}\text{.} \end{equation*}
    üîó
    The additive inverse of \(\frac{a}{b}\) is \(\frac{-a}{b}\) as:
    \begin{equation*} \frac{a}{b} + \frac{-a}{b} = \frac{ab+b(-a)}{b^2} = \frac{0}{b^2}. \end{equation*}
    And \(\frac{0}{1} = \frac{0}{b^2}\) as \(0\cdot b^2=1\cdot 0\text{.}\)
    üîó
    We now turn to the multiplicative axioms: associativity, commutativity, identity, inverses of non-zero elements. For associativity:
    \begin{align*} \frac{a}{b}\cdot\left(\frac{c}{d}\cdot \frac{e}{f}\right) \amp= \frac{a}{b}\cdot \frac{ce}{df} \\ \amp= \frac{a(ce)}{b(df)}\\ \left(\frac{a}{b}\cdot\frac{c}{d}\right) \cdot \frac{e}{f} \amp= \frac{ac}{bd}\cdot \frac{e}{f} \\ \amp= \frac{(ac)e)}{(bd)f}\\ \amp= \frac{a(ce)}{b(df)} \end{align*}
    where the last line uses associativity of multiplication in \(\Z\text{.}\)
    üîó
    For commutativity we again use that the property holds in \(\Z\text{:}\)
    \begin{equation*} \frac{a}{b}\cdot \frac{c}{d} = \frac{ac}{bd} = \frac{ca}{db} = \frac{c}{d}\cdot \frac{a}{b}. \end{equation*}
    üîó
    The multiplicative identity is \(\frac{1}{1}\) as
    \begin{equation*} \frac{1}{1}\cdot \frac{a}{b} = \frac{1\cdot a}{1\cdot b} = \frac{a}{b}. \end{equation*}
    üîó
    Lastly we have the left and right-distributive axioms. For the left-distributive axiom we have:
    \begin{align*} \frac{a}{b}\cdot\left(\frac{c}{d}+\frac{e}{f}\right) \amp= \frac{a}{b}\cdot \frac{cf+de}{df} \\ \amp=\frac{a(cf+de)}{b(df)} \\ \amp= \frac{acf+ade}{bdf} \\ \frac{a}{b}\cdot \frac{c}{d} + \frac{a}{b}\cdot \frac{e}{f} \amp= \frac{ac}{bd} + \frac{ae}{bf} \\ \amp= \frac{acbf+bdae}{b^2df} \end{align*}
    Now notice that \(\frac{acbf + bdae}{b^2df} = \frac{acf+ade}{bdf}\) as
    \begin{equation*} (acbf+bdae)(bdf) = b^2df(acf+ade). \end{equation*}
    üîó
    It is a consequence of the axioms that left and right-distributivity are equivalent, see Exercise¬†2.1.7.10. We show how to prove this exercise in this example:
    \begin{align*} \left(\frac{a}{b} + \frac{c}{d}\right)\cdot \frac{e}{f}\amp= \frac{e}{f}\cdot\left(\frac{a}{b} + \frac{c}{d}\right) \\ \amp= \frac{e}{f}\cdot \frac{a}{b} + \frac{e}{f}\cdot \frac{c}{d} \\ \amp= \frac{a}{b}\cdot \frac{e}{f} + \frac{c}{d}\cdot \frac{e}{d} \end{align*}
    where the first and third equality is commutativity of multiplication and the second is the left-distributive property.
    üîó
    üîó
    This was quite involved but we managed it. This was our first check that a set with one or more binary operations satisfied a sequence of axioms. These types of checks will continue throughout the course and so we will spend quite some time developing procedural fluency in carrying out definition checks.

    Example 2.1.7. Complex Numbers Form a Field.

    The Complex Numbers Definition¬†2.1.3 form a field under their usual multiplication and addition. The operations are well-defined and so now we need to check all ten axioms. We leave a handful of them to the reader as they are not necessarily difficult but they are tedious. Take \(a+bi, c+di, e+fi\in\C\text{.}\)
    üîó
    We leave the proof of associativity as an exercise, as well as the proof that the additive identity is \(0+0i\) and the additive inverse of \(a+bi\) is \((-a)+(-b)i\text{.}\) We will prove that addition is commutative:
    \begin{align*} (a+bi) + (c+di) \amp= (a+c) + (b+d)i \\ (c+di) + (a+bi) \amp= (c+a) + (d+b)i \\ \amp= (a+c) + (b+d)i \end{align*}
    as \(a,b,c,d\in \R\) and addition of real numbers is commutative.
    üîó
    We leave the proof of associativity as a worthy yet tedious exercise. We will prove the remaining axioms. For commutativity:
    \begin{align*} (a+bi)\cdot(c+di) \amp= (ac-bd) + (ad+bc)i \\ (c+di)\cdot (a+bi) \amp= (ca-db) + (cb+da)i \\ \amp= (ac-bd) + (ad+bc)i \end{align*}
    where have used the multiplication of real numbers is commutative.
    üîó
    The multiplicative identity is \(1+0i\text{:}\)
    \begin{equation*} (a+bi)\cdot(1+0i) = (a\cdot 1-b\cdot 0) + (a\cdot 0 + b\cdot 1)i = a+bi \end{equation*}
    üîó
    The multiplicative inverse of a non-zero \(a+bi\) is \(\frac{a}{a^2+b^2} + \frac{-b}{a^2+b^2}i\text{.}\) Since \(a+bi\neq 0+0i\text{,}\) \(a^2+b^2>0\) and so this complex number is well-defined. We leave it as an exercise to verify:
    \begin{equation*} (a+bi)\cdot \left(\frac{a}{a^2+b^2} + \frac{-b}{a^2+b^2}i\right) = 1 + 0i \end{equation*}
    üîó
    We‚Äôve already seen an instance of the left-distributive property implying the right-distributive property. Since you will prove this in Exer???? we only show the left-distributive property:
    \begin{align*} \amp (a+bi)\cdot( (c+di) + (e+fi))\\ \amp= (a+bi)\cdot( (c+e)+(d+f)i)\\ \amp= (a(c+e)-b(d+f)) + (a(d+f)+b(c+e))i\\ \amp= (ac+ae-bd-bf) + (ad+af+bc+be)i \end{align*}
    and
    \begin{align*} \amp (a+bi)\cdot (c+di) + (a+bi)\cdot (e+fi)\\ \amp= ((ac-bd)+(ad+bc)i) + ((ae-bf) + (af+be)i)\\ \amp= (ac-bd) + (ae-bf) + ( (ad+bc) + (af+be))i\\ \amp= (ac+ae-bd-bf) + (ad+af+bc+be)i \end{align*}
    These agree and completes the proof of the left-distributive property and that \(\C\) is a field.
    üîó
    üîó
    The next example is a finite example. In particular, we will return to Definition¬†1.4.5 and prove that the integers modulo \(m\) form a field if and only if \(m\) is prime.
    üîó
    Before the proof, we recall that addition and multiplication is associative and commutative was done in exercises ????. The additive identity is \(\bar{0}\) as \(\bar{0}+\bar{a} = \overline{0+a} = \bar{a}\text{.}\) The additive inverse of \(\bar{a}\) is \(\overline{-a}\text{.}\) The multiplicative identity is \(\bar{1}\) as \(\bar{1}\cdot \bar{a} = \overline{1\cdot a} = \bar{a}\text{.}\) That multiplication distributes over addition was also done in exercise???.
    üîó
    In particular, \(\Z/m\) satisfies all the axioms of being a field with the exception of multiplicative inverses for non-zero elements. In the proof that follows we show this happens if and only if \(m\) is prime.
    üîó

    Theorem 2.1.8.

    Fix \(p>1\) in \(\N\text{.}\) Then \(\Z/p\text{,}\) with modular addition and multiplication, is a field if and only if \(p\) is prime.
    üîó
    üîó

    Proof.

    By the discussion preceeding the theorem statement, we need only check that multiplicative inverses exist to see that \(\Z/p\) is a field. To that end, suppose \(p\) is prime. Take \(\bar{0}\ne \bar{a}\in\Z/p\text{.}\) Since \(p\) is prime, \(\gcd(a,p) = 1,p\text{.}\) If \(\gcd(a,p) = p\text{,}\) then \(\bar{a} = \bar{0}\) which isn‚Äôt true. Thus \(\gcd(a,p) = 1\text{.}\) By Theorem¬†1.2.9, there exists \(x,y\in\Z\) with
    \begin{equation*} ax+py=1. \end{equation*}
    üîó
    Since \(\bar{p} = \bar{0}\text{,}\) after reducing modulo \(p\) we have
    \begin{equation*} \bar{a}\cdot \bar{x} = \bar{1} \end{equation*}
    and in particular \(\bar{x}\) is the multiplicative inverse of \(\bar{a}\text{.}\)
    üîó
    If \(p\) is not prime, then there exists \(a,b\in\mathbb{N}\) with \(p=ab\) and \(1\le a,b\le p\text{.}\) Then \(\bar{a}\cdot \bar{b} = \bar{p} = \bar{0}\text{.}\) And since \(1\leq a,b\leq p\text{,}\) we have \(\bar{a},\bar{b}\neq \bar{0}\text{.}\) If \(\Z/p\) was a field, then \(\bar{a}\) has a multiplicative inverse, say \(\bar{a}^{-1}\text{.}\) Then
    \begin{align*} \bar{a}\bar{b} \amp= \bar{0} \\ \bar{a}^{-1}\cdot(\bar{a}\bar{b}) \amp= \bar{a}^{-1}\cdot \bar{0} \\ (\bar{a}^{-1}\cdot\bar{a})\bar{b} \amp= \bar{0} \\ \bar{1}\cdot \bar{b}\amp= \bar{0} \\ \bar{b}\amp= \bar{0} \end{align*}
    which contradicts \(\bar{b}\ne \bar{0}\text{.}\) We conclude the non-zero element \(\bar{a}\) does not have a multiplicative inverse and so when \(p\) is not prime \(\Z/p\) is not a field.
    üîó
    üîó
    Notice that when \(p\) is prime \(\Z/p\) is a field and it has only finitely many elements. This is such an important field, we will give a proper definition.
    üîó

    Definition 2.1.9.

    For \(p\) a prime, we denote by \(\mathbb{F}_p\) the finite field \(\Z/p\) and call it the finite field of order \(p\text{.}\)
    üîó
    üîó
    The symbol \(\mathbb{F}_p\) means the same as \(\Z/p\) in terms of field arithmetic. The notation \(\mathbb{F}_p\) is often used to emphasize that we are thinking about \(\Z/p\) as a field.
    üîó

    Subsection 2.1.5 Properties of Fields

    Theorem 2.1.10. Properties of Fields.

    Let \(F\) be a field.
    1. The additive identity \(0\) is unique.
      üîó
      üîó
    2. For all \(a\in F\text{,}\) \(a \cdot 0 = 0\cdot a = 0\text{.}\)
      üîó
      üîó
    3. Additive inverses are unique.
      üîó
      üîó
    4. The multiplicative identity \(1\) is unique.
      üîó
      üîó
    5. Multiplicative inverses are unique.
      üîó
      üîó
    6. \(\displaystyle (-1)\cdot (-1) = 1\)
      üîó
      üîó
    üîó
    üîó

    Proof.

    1. Assume \(0, 0'\) both satisfy the additive identity axiom. Observe that \(0 = 0 + 0' = 0'\text{.}\)
      üîó
      üîó
    2. As \(0 = 0 + 0\text{,}\) we may write \(a\cdot 0 = a\cdot (0 + 0) = a\cdot 0 + a \cdot 0\text{.}\) Now add the additive inverse of \(a\cdot 0\) to both sides to obtain \(0 = a \cdot 0 = 0\cdot a\text{.}\)
      üîó
      üîó
    3. Let \(a\in F\) and suppose \(b\) and \(c\) are such that \(a+c = 0\) and \(a+b = 0\text{.}\) Then \(a+c = a+b\text{,}\) and we may add \(b\) (or \(c\)) to both sides to obtain \(b+(a+b) = (b+a)+b = 0 + b = b\) and \(c+(a+c) = (c+a)+c = 0 + c = c\text{.}\) Thus \(b=c\text{.}\)
      üîó
      üîó
    4. Suppose \(1\) and \(1'\) are multiplicative identities. Then \(1 = 1 \cdot 1' = 1'\text{.}\)
      üîó
      üîó
    5. Let \(a\in F\) be nonzero and suppose \(b,c\) are multiplicative inverses for \(a\text{.}\) Then \(ab = 1 = ac\text{,}\) and we may multiply by \(b\) (or \(c\)) to obtain \(b(ab) = (ba)b = 1\cdot b = b = b(ac) = (ba)c = 1\cdot c = c\text{.}\)
      üîó
      üîó
    6. Observe that \((-1) + 1 = 0\) by the definition of additive inverses. Multiply both sides by \(-1\) and distribute to obtain \((-1)(-1) + (-1)\cdot 1 = (-1)\cdot 0\text{.}\) By part 2 of this theorem and the definition of a multiplicative identity, we obtain \((-1)(-1) + (-1) = 0\text{.}\) We now add the multiplicative identity to obtain \((-1)(-1) + (-1) + 1 = 0 + 1\text{,}\) which simplifies to \((-1)(-1) = 1\text{.}\)
      üîó
      üîó
    üîó
    üîó
    One consequence of Theorem¬†2.1.10 is that, given \(a\in F\text{,}\) \(b\in F\setminus \set{0}\text{,}\) we may refer to \(-a\) as the additive inverse of \(a\text{,}\) and \(b^{-1}\) as the multiplicative inverse of \(b\text{.}\) We will thus employ this familiar terminology henceforth. Moreover, we will define subtraction in a field as \(a-b = a+(-b)\) and division as \(a/b = a\cdot b^{-1}\text{.}\)
    üîó
    üîó

    Subsection 2.1.6 Subfields and Extensions

    One source for new fields is to look within the fields we already have. That is, we want to know when a subset of a field is a field in its own right.
    üîó

    Definition 2.1.11.

    Let \(K\) be a field. If \(F\) is a nonempty subset satisfying
    1. \(F\subseteq K\) and
      üîó
    2. \(F\) is a field under the same operations as \(K\text{,}\)
      üîó
    üîó
    then we call \(F\) a subfield of \(K\text{,}\) and \(K\) an extension field of \(F\text{,}\) and say ‚Äú\(K/F\) is a field extension.‚Äù
    üîó
    üîó
    Many of the number sets with which you are familiar are fields.
    üîó

    Theorem 2.1.12.

    The set of complex numbers \(\mathbb{C}\) is an extension field of \(\R\text{.}\) Equivalently, \(\R\) is a subfield of \(\mathbb{C}\)
    üîó
    üîó
    The last theorem in this section gives us a technique for identifying subfields of a given field.
    üîó

    Theorem 2.1.13. Subfield Test.

    Let \(K\) be a field and \(F\subseteq K\) be a subset of \(K\) such that:
    • \(\displaystyle 1_K\in F\)
      üîó
      üîó
    • For any \(a,b\in F\text{,}\) we have \(a-b\in F\text{.}\) (Closed under subtraction)
      üîó
      üîó
    • For any \(a,b\in F\text{,}\) we have \(a/b\in F\text{.}\) (Closed under division)
      üîó
      üîó
    Then \(F\) is a field under the addition and multiplication in \(K\text{.}\) In particular, \(F\) is a subfield of \(K\text{.}\)
    üîó
    üîó

    Proof.

    We check all of the axioms in Definition¬†2.1.5. Since \(F\) is closed under subtraction, \(1_K-1_K=0_K\in F\text{.}\) Since \(0_K\in F\) and \(F\) is closed under subtraction, we have \(-a=0_K-a\in F\) for any \(a\in F\text{.}\) Thus \(F\) is closed under additive inverses. Since \(F\) is closed under additive inverses and subtraction we have \(a+b = a-(-b)\in F\) for any \(a,b\in F\text{.}\) Thus \(F\) is closed under addition.
    üîó
    We now turn to the associativity and commutative properties. For any \(a,b,c\in F\) we know \(a,b,c\in K\text{.}\) Since addition in \(K\) is associative, we know \(a+(b+c) = (a+b)+c\text{.}\) Thus addition in \(F\) is associative. We say it is inherited from \(K\text{.}\) Similarly addition in \(F\) is commutative, \(0_K\) is the additive identity, multiplication is associative and commutative, \(1_K\) is the multiplicative identity, and the distributive laws hold.
    üîó
    Lastly, since \(F\) is closed under division and \(1_K\in F\text{,}\) for any \(b\in F\) we have \(1_K/b\in F\) and so the multiplicative inverse of \(b\) is in \(F\text{.}\) Thus \(F\) is closed under taking multiplicative inverses. This completes the checks of each of the 10 axioms.
    üîó
    üîó
    We end this section with another interesting example of a field. We also illustrate how to use the Subfield Test.
    üîó

    Example 2.1.14. Adjoining \(\sqrt{-5}\) to the Rationals.

    Define the set of rationals adjoin \(\sqrt{-5}\) by:
    \begin{equation*} \Q(\sqrt{-5}) = \{a+b\sqrt{-5}\mid a,b\in\mathbb{Q}\}. \end{equation*}
    This is a subset of \(\mathbb{C}\text{.}\) We claim that \(\Q(\sqrt{-5})\) is a subfield of \(\mathbb{C}\)
    üîó
    We apply the Subfield Test. In particular, \(\mathbb{Q}(\sqrt{-5})\subset \C\) and the latter is a field. We first show that \(1+0i\in\Q(\sqrt{-5})\) but this is just the element \(1+0\sqrt{d}\in\Q(\sqrt{-5})\text{.}\)
    üîó
    We now show closure under subtraction. Take \(a+b\sqrt{-5},c+d\sqrt{-5}\in\Q\text{,}\) then
    \begin{equation*} (a+b\sqrt{-5}) - (c+d\sqrt{-5}) = (a-c) + (b-d)\sqrt{-5}. \end{equation*}
    Since \((a-c),(b-d)\in \Q\text{,}\) we conclude that \(\Q(\sqrt{-5})\) is closed under subtraction.
    üîó
    We now show closure under division. Suppose \(a+b\sqrt{-5},c+d\sqrt{-5}\in\Q(\sqrt{-5})\text{.}\) Writing \(c+d\sqrt{-5} = c+d\sqrt{5}i\text{,}\) we know that
    \begin{equation*} (c+d\sqrt{5}i)^{-1} = \frac{c}{c^2+5d^2} + \frac{-d\sqrt{5}}{c^2+5d^2}i \end{equation*}
    which we can recombine the \(i\) and \(\sqrt{5}\text{:}\)
    \begin{equation*} (c+d\sqrt{5}i)^{-1} = \frac{c}{c^2+5d^2} + \frac{-d}{c^2+5d^2}\sqrt{-5}. \end{equation*}
    Note that \(c^2+5d^2=0\) would say \((c/d)^2 = -5\) which is absurd so the inverse is well-defined.
    üîó
    Now we divide:
    \begin{align*} \amp(a+b\sqrt{-5})\cdot (c+d\sqrt{-5})^{-1} \\ =\amp(a+b\sqrt{-5})\cdot \left(\frac{c}{c^2+5d^2} + \frac{-d}{c^2+5d^2}\sqrt{-5} \right) \\ =\amp \left(\frac{ac}{c^2+5d^2} + \frac{5bd}{c^2+5d^2}\right) + \left(\frac{-ad}{c^2+5d^2}+ \frac{bc}{c^2+5d^2}\right)\sqrt{-5}\\ =\amp \frac{ac+5bd}{c^2+5d^2} + \frac{-ad+bc}{c^2+5d^2}\sqrt{-5} \end{align*}
    and since \(\Q\) is a field it is closed under arithmetic. In particular \(\frac{ac+5bd}{c^2+5d^2}, \frac{-ad+bc}{c^2+5d^2}\in \Q\text{.}\) Thus \(\Q(\sqrt{-5})\) is closed under division.
    üîó
    We conclude, by the Subfield Test, that \(\Q(\sqrt{-5})\) is a subfield of \(\C\text{.}\)
    üîó
    üîó
    The reader should take note that the importance of using \(-5\) under the radical was only that \(\sqrt{-5}\notin \Q\text{.}\) In particular, for any \(d\in\Q\) such that \(\sqrt{d}\notin\Q\) the proof will go through exactly the same. You will carry this out in Exercise????
